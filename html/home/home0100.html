Natural-JS is a Javascript library that can be easily and quickly development the enterprise web application UI such as ERP, CRM, etc.

Natraul-JS 의 특징

Features of Natural-JS

  • CVC 아키텍처 패턴 기반의 UI 아키텍처와 이를 지원하는 라이브러리를 제공합니다.

    Provides the CVC architecture pattern based architecture and libraries for support to UI architecture

    • 개발영역과 디자인영역을 완벽하게 분리하여 서버 개발자와 UI 개발자, 또는 스크립트 개발자와 디자이너간의 업무 분쟁을 최소화 할 수 있습니다. 또한 이로 인해 서버 개발 조직과 UI 개발 조직을 분리하여 웹 어플리케이션 개발 프로젝트를 진행 할 수 있습니다.
    • It can completely separate the development and design areas and minimize disputes between server side developers and UI side developer or script developer and designer. In addition, As a result, that may separate the server side and UI side development organization during to do web application development projects.
    • 여러개의 블록페이지(View와 Controller로 구성된 단위 페이지)들을 조합하여 하나의 페이지로 완성할 수 있습니다. 이때 스크립트간, 요소간 Scope 을 완벽하게 분리 해 줍니다.
    • You can combine multiple block pages (page Units consisting of View and Controller) to complete a single page. at this time, it helps ensure full separation between script and elements scope.
    • 단위 페이지별 개발영역의 틀을 정의 할 수 있는 Controller(N.cont) 라이브러리를 제공 합니다.
    • Provides the Controller(N.cont) library that can be help to define the frame of the development area per block page
    • UI(클라이언트 웹 브라우저) 와 서버간 데이터 송수신 역할을 담당하는 Communicator(N.comm) 라이브러리를 제공 합니다.
    • Provides the Communicator(N.comm) library that can be transmitting and receiving data between the UI(Client Web Browser) side and server side.

  • 데이터의 조작과 가공, 입력된 데이터의 검증을 편리하게 할 수 있는 라이브러리를 제공합니다.

    Provides the libraries for easy control that supports to manipulation and processing of the data, verification of the input data

    • 데이터에 대한 양식(Format)을 손쉽게 변환 할 수 있는 Formatter 라이브러리를 제공합니다.
    • Provides the Formatter library that can be easily convert the format of the data.
    • 데이터에 대한 검증(Validation)을 손쉽게 수행 할 수 있는 Validator 라이브러리를 제공합니다.
    • Provides the Validator library that can be easily validate to the input data.
    • 간단하게 데이터를 조작하거나 가공 할 수 있는 라이브러리를 제공 합니다.
    • Provides the libraries for simply manipulating and processing the data.

  • Rich-UI를 표현하기 위한 다양한 UI 컴포넌트를 제공 합니다.

    Provides the various UI-Components for express the Rich-UI.

    • 지정한 요소에 데이터 바인드를 손쉽게 할 수 있는 Grid, Form, Select, Pagination, Tree 컴포넌트를 제공합니다.
    • Provides the Grid, Form, Select, Pagination and Tree components that can be easily bind the data to the selected elements
    • Rich UI를 쉽게 구현할 수 있도록 해 주는 Alert, Popup, Tab, Button, Datepicker 컴포넌트를 제공합니다.
    • Provides the Alert, Popup, Tab, Button and Datepicker components that can be easily development for Rich-UI
    • 조회하거나 생성한 데이터와 HTML 요소의 값을 따로 관리하고 이를 동기화 시켜 줍니다.
    • Queried or generated data and HTML elements to be separately managed and these are synchronized
    • 각각의 UI 컴포넌트들은 실시간으로 데이터가 동기화(양방향 바인드)되고 상호작용을 합니다.
    • Data of the each UI-Components are synchronized on real time(Two way bind) and that is interacted each other.
    • 각각의 컴포넌트는 퍼블리싱된 HTML 템플릿의 모양과 디자인을 최대한 유지시켜 줍니다.
    • Each components are keeps most the shape and graphic design of be published HTML templete

    • Grid, Form 등의 컴포넌트는 자체 스타일을 가지고 있지 않습니다. 예를 들어, Grid 컴포넌트를 사용하는 경우, 페이지 내 table 태그에 대한 스타일이 공통 css나 직접 css를 지정하여 적용되어 있다고 하면 Grid 컴포넌트 적용 시 정의된 모양을 최대한 유지 시켜 컴포넌트 화 해 줌으로서 컴포넌트 디자인을 사이트 디자인 컨셉에 간편하게 적용할 수 있습니다.

    • Components such as Grid and Form do not have their own style. For example, if you are use the Grid component, If the styles for the table tags in the page are applied by specifying common css or direct css, You can easily apply component design to site design concepts by maintaining the defined table's style when applying the Grid component

Natural-JS 의 구조
Structure of Natural-JS

Natural-JS 는 Natural-CORE, Natural-ARCHITECTURE, Natural-DATA, Natural-UI 로 구성 됩니다.

Natural-JS is composed of a Natural-CORE, Natural-ARCHITECTURE, Natural-DATA, Natural-UI.

 

Natural-CORE는 프레임워크에서 내부적으로 사용하는 CORE 라이브러리 패키지이고
Natural-ARCHITECTURE는 복잡한 웹 UI 환경을 구조화, 단순화 시켜주는 CVC Architecture Pattern 기반의 Architecture Framework를 제공 합니다.
Natural-DATA는 Data 처리 및 데이터 동기화에 대한 모듈들을 제공 하며
Natural-UI 는 HTML 기반의 Grid, Form, Popup등 여러 가지 컴포넌트를 제공 합니다.

Natural-CORE is CORE library package that is used internally by the framework. Natural-ARCHITECTURE is CVC Architecture Pattern based Architecture Framework. it makes structured and simplifies a complex web UI environment. Natural-DATA provides a module for Data processing and data synchronization. Natural-UI provides a variety of components such as HTML-based Grid, Form, Popup that help improve the user experience and web development productivity.
