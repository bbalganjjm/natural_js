Controller(N.cont)

    개요 Overview

    Controller(이하 N.cont) 는 CVC Architecture Pattern(이하 CVC 패턴)의 Controller 영역을 구현한 클래스로서 Communicator 와 View 를 연결 해 주고 View 의 HTML 요소(element)들을 제어하는 역할을 합니다.

    N.cont 는 다음과 같이 페이지의 View 영역 바로 아래 선언 되어야 합니다.

    <div id="view">
	<p>View 영역</p>
</div>

<script type="text/javascript">
	N("#view").cont({
		init : function(view, request) {
			// 페이지 로딩 후 최초 실행되는 영역
		}
	});
</script>

    위와 같이 N.cont 를 선언하면 N.comm이나 N.popup, N.tab 등으로 페이지를 호출 했을 경우 View 영역의 Load 가 완료된 후 N.cont의 인자로 들어온 Controller 클래스에 init 메서드가 정의 되어 있다면 init 메서드를 프레임워크에서 실행 시켜 줍니다. init 의 인자(Arguments)로는 View영역의 최 상위 요소가 선택된 jquery object 객체를 첫번째 인자로 반환하고 Communicator.request(Communicator 참조) 객체를 두번째 인자로 반환 합니다. 또한 Controller Class 의 초기화(Initialize)가 완료되면 View영역의 최 상위 Element 에 해당 Controller 인스턴스(instance)를 넣어줘 N("페이지 최 상위 Element").instance("cont") 로 View 영역에서 참조 할 수 있습니다.

    Natural-ARCHITECTURE 기반으로 구성된 페이지들은 무조건 N.comm 이나 N.popup, N.tab 으로 불러와야 정상적으로 작동 됩니다.

    생성자 Constructor

    Name Arguments Type Return Description
    N.cont - - (Controller)callback

    Controller를 초기화 합니다.

    N.cont는 컨트롤러 클래스를 등록하고 관리 해 주는 역할을 수행 합니다.
    obj jquery object - View 영역을 selector로 지정 합니다.
    callback json object class - Controller 클래스를 등록 합니다.
    N(selector).cont - - -

    N.cont를 플러그인 방식으로의 초기화 방식입니다.

    N.cont 와 동작되는 로직은 같으며 실제 N() 의 인자로 들어온 값이 N.cont()의 첫 번째 인자로 지정 됩니다.

    이와같은 방식으로 사용하면 컴포넌트를 초기화하는 여러 가지 설정들을 자동으로 세팅해 주므로 Natural-JS 에서는 이와같은 방식의 사용을 추천합니다.
    callback object Class - N.cont 의 callback 인자(argument) 와 같습니다.

    Controller(N.cont) 객체 변수 Controller(N.cont) object's variables

    Name Type Description
    view jquery object view 요소가 정의 되어 있습니다.
    request Communicator.request 현재 페이지의 요청 정보와 페이지 호출 시 전달 한 데이터를 얻을 수 있는 Communicator.request 객체가 정의 되어 있습니다.
    caller N.popup 팝업으로 호출 된 페이지 일 경우 자신을 호출 한 팝업 인스턴스가 정의 되어 있습니다.
    opener object

    팝업으로 호출 된 페이지 일 경우 자신을 호출 한 팝업 인스턴스가 포함된 컨트롤러 오브젝트가 정의 되어 있습니다.

    팝업 인스턴스가 무기명이거나 지역 변수일 경우 opener 속성이 정의 되지 않을 수 있습니다. 먼저 opener 속성이 undefined 인지 체크 후 사용 바랍니다.

    Controller(N.cont) 객체 함수 Controller(N.cont) object's functions

    Name Arguments Type Return Description
    init - - user define N.cont 에 의해 등록된 callback 이 초기화 된 후 최초 실행 되는 함수입니다.
    view jquery object -

    view 요소를 반환 합니다.
    request Communicator.request - 현재 페이지의 요청 정보와 페이지 호출 시 전달 한 데이터를 얻을 수 있는 Communicator.request 객체를 반환 합니다.

    예제 Examples

    1. 뷰(view)와 컨트롤러 정의

    
<!-- View area start -->
<div id="view">
	<p>View 영역</p>
</div>
<!-- View area end -->

<script type="text/javascript">
	// Controller area start
	N("#view").cont({
		init : function(view, request) {
			// 페이지 로딩 후 최초 실행되는 영역
		}
	});
	// Controller area end
</script>
