Natural-JS

Natural-JS는 기업용 웹 어플리케이션 UI 를 쉽고 빠르게 구현할 수 있는 오픈소스 Javascript 프레임워크입니다.

Natural-JS is an open source Javascript framework that can be easily and quickly development the enterprise web application UI.

Natural-JS의 특징

Features of Natural-JS

CVC 아키텍처패턴 기반의 Natural-ARCHITECTURE 프레임워크를 제공합니다.

Provides an Natural-ARCHITECTURE framework based on the CVC architecture pattern.

  • AOP

    Controller object의 사용자 정의 함수에 AOP(Aspect-Oriented Programming)를 적용할 수 있습니다.

  • AOP

    You can apply AOP(Aspect-Oriented Programming) to user-defined functions of the Controller object.

  • Communication Filter

    서버와 주고받는 모든 데이터를 필터링할 수 있는 Communication Filter 기능을 제공합니다.

  • Communication Filter

    It provides Communication Filter that can filter all data sent/receive with the server.

  • 웹 컴포넌트

    페이지를 블록단위로 개발할 수 있는 아키텍처를 제공합니다. 이 페이지 블록들은 Tab이나 Popup의 컨텐츠나 SPA(Single Page Application)의 페이지 컨텐츠, MSA(Micro Service Architecture)의 웹 컴포넌트로 로드할 수 있습니다.

  • Web Components

    Provides an architecture for developing pages on a block-by-block basis. These page blocks can be loaded into Tab or Popup contents, page content of Single Page Application(SPA), or Web Component of Micro Service Architecture(MSA).

  • UI 개발의 분업화

    UI 소스코드 안에서 개발 영역과 프리젠테이션 영역을 완벽하게 분리해 주어 웹 퍼블리셔와 스크립트 개발자 또는 Back-end 개발자와 Front-end 개발자의 역할을 분리하여 프로젝트를 진행할 수 있습니다.

  • Division of labor for UI development

    In the UI source code, the development area and the presentation area are completely separated so that the roles of the web publisher and script developer or the back-end developer and front-end developer can be separated to proceed with the project.

데이터 표현과 컨트롤, Rich-UI 개발을 위한 다양한 UI 컴포넌트를 제공합니다.

It provides various UI components for data presentation, control, and rich-UI development.

  • 데이터 처리 컴포넌트

    데이터를 요소에 바인딩하거나 바인딩된 데이터를 제어할 수 있는 Grid, List, Form, Select, Pagination, Tree 컴포넌트를 제공합니다.

  • Data Processing Components

    Provides Grid, List, Form, Select, Pagination, and Tree components that can bind data to elements or control bound data.

  • UI 컴포넌트

    Rich-UI 개발을 지원하는 Alert, Popup, Tab, Button, Datepicker, Notify, Documents 컴포넌트를 제공합니다.

  • UI Components

    Provides Alert, Popup, Tab, Button, Datepicker, Notify, Documents components that support Rich-UI development.

  • 양방향 데이터 바인딩

    서로 다른 UI 컴포넌트에 같은 데이터를 바인딩하면 데이터가 실시간으로 동기화되고 상호작용을 합니다.

  • Two Way Databinding

    Binding the same data to different UI components synchronizes and interacts in real time.

  • 컴포넌트 디자인

    컴포넌트 스타일을 쉽게 변경할 수 있고 사이트의 공통 스타일이 컴포넌트에 자동으로 적용되게할 수 있습니다.

  • Component Design

    Component styles can be easily changed and the common style of the site can be automatically applied to components.

데이터를 손쉽게 핸들링할 수 있는 라이브러리를 제공합니다.

Provides a library for easy handling of data.

  • 데이터 서식화

    데이터 값의 서식을 손쉽게 지정 할 수 있는 Formatter 라이브러리를 제공합니다.

  • Data Formatting

    Provides the Formatter library that allows you to easily format data values.

  • 데이터 유효성 검증

    입력값의 유효성을 손쉽게 검증(Validate)할 수 있는 Validator 라이브러리를 제공합니다.

  • Data Validation

    Provides the Validator library that allows you to easily validate input values.

  • 데이터 유틸리티

    데이터를 조작하거나 가공할 수 있는 filter, sort 등의 라이브러리를 제공합니다.

  • Data Utilities

    Provides libraries such as filters and sort that can manipulate or process data.

모바일 브라우저(iOS Safari, iOS UIWebView, Android Browser, Android Chrome, Android WebView)와 터치 디바이스를 지원하고 모바일 하이브리드 앱과의 연동을 지원합니다.

Supports mobile browsers (iOS Safari, iOS UIWebView, Android Browser, Android Chrome, Android WebView) and touch devices and it supports interworking with mobile hybrid applications.

Natural-JS의 구성

Structure of Natural-JS

  • 프레임워크 공통 라이브러리 패키지

    Framework Common Library Package

  • 복잡한 웹 UI 환경을 구조화, 단순화켜주는 아키텍처 프레임워크 패키지

    Architecture framework package to structure and simplify complex web UI environment

  • Data 처리 및 데이터 동기화와 관련된 확장 모듈 패키지

    Expansion module package related to data processing and data synchronization

  • HTML 기반의 UI 컴포넌트 패키지

    HTML-based UI component package
