N.comm N/A N/A N.comm

N.comm 인스턴스를 생성합니다.

Create an N.comm instance.

N.comm은 new 연산자로 인스턴스를 생성하지 않고 N.comm객체를 직접 사용할 수 있습니다.

N.comm can use N.comm objects directly without instantiating them with the new operator.

N.comm 함수가 호출되거나 new 연산자로 N.comm 인스턴스를 생성하면 객체의 request 속성 값에 Communicator.request 객체 인스턴스가 생성되어 바인딩됩니다.

If the N.comm function is called or the N.comm instance is created with the new operator, the Communicator.request object instance is created and bound to the request property value of the object.

var comm = N.comm(param|element|url, opts|url);
param|element|url jQuery object[array[object]]|jQuery object[object]|jQuery object[element]|object|string N/A

첫 번째 인자에 object 나 array[object] 타입의 객체가 들어있는 jQuery object로 설정하고 두 번째 인자(url)에 JSON 데이터를 반환하는 서버 URL을 설정하면 설정된 객체를 문자열로 변환 한 후에 서버에 GET이나 POST 파라미터로 전달합니다. 이때 이 인자값은 Communicator.request의 기본 옵션의 data속성 값으로 설정됩니다.

If you set a jQuery object that contains an object of type object or array [object] in the first argument, and set a server URL that returns JSON data in the second argument (url), it converts the set object to a string and then passes it to the server as a GET or POST parameter. At this time, the value of this parameter is set as the value of the data property of the default option of Communicator.request.

N.comm({ "param" : "value" }, "data.json").submit(function(data) {});

type(HTTP METHOD) 옵션이 GET 일 경우 data속성 값(param)이 object 이거나 array[object] 타입이면 "q" 파라미터 키에 URL encoding된 파라미터 문자열이 값으로 설정됩니다.
  ex) data.json?q=%5B%7B%22param%22%3A%22value%22%7D%5D // q=[{"param":"value"}]
GET 파라미터는 전송할 수 있는 데이터의 최대값이 있기 때문에 브라우저별 GET 파라미터의 최대 전송량을 파악하여 사용하기 바랍니다.

If the type (HTTP METHOD) option is GET, if the data attribute value (param) is object or array [object] type, the URL-encoded parameter string is set as the value in the "q" parameter key.
  ex) data.json?q=%5B%7B%22param%22%3A%22value%22%7D%5D // q=[{"param":"value"}]
Since the GET parameter has the maximum value of the data that can be transferred, please check and use the maximum amount of GET parameters for each browser.

첫 번째 인자에 HTML 요소가 담겨있는 jQuery object 타입을 설정하고 두 번째 인자(url)에 HTML 페이지를 반환하는 서버 URL을 설정하면 로드된 페이지가 지정된 요소에 삽입됩니다.

If you set the jQuery object type that contains the HTML element in the first argument and the server URL that returns the HTML page in the second argument (url), the loaded page is inserted to the specified element.

N.comm(N("#contents"), "page.html").submit();

페이지에 파라미터를 전달하는 경우에는 Communicator.request 객체의 attr 메서드를 사용하세요.

If passing parameters to a page, use the attr method of the Communicator.request object.

N.comm(N("#contents"), "page.html").request.attr("pageParam", "value").submit();

object 타입을 설정하면 Communicator.request 객체의 기본 옵션으로 설정됩니다.

If object type is set, it is set as the default option for Communicator.request objects.

N.comm({ "param" : "value" }, {
    url : "data.json"
}).submit(function(data) {});

string 타입을 설정하면 Communicator.request 객체의 기본 옵션의 url옵션으로 설정됩니다.

If string type is set, it is set as url option of default option of Communicator.request object.

N.comm("data.json").submit(function(data) {});
opts|url object|string N/A

object 타입을 설정하면 Communicator.request 객체의 기본 옵션으로 설정됩니다.

If object type is set, it is set as the default option for Communicator.request objects.

string 타입을 설정하면 Communicator.request 객체의 기본 옵션의 url옵션으로 설정됩니다.

If string type is set, it is set as url option of default option of Communicator.request object.
N(obj).comm N/A N/A N.comm

N.comm의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

Create an object instance of N.comm with jQuery plugin method.

var comm = N(param|element).comm(opts|url);

객체 인스턴스를 생성하는 방식은 다르지만 "new N.comm()"로 생성한 인스턴스와 "N().comm"로 생성한 인스턴스는 동일합니다. N() 함수의 첫 번째 인자가 new N.comm 생성자의 첫 번째 인자로 설정됩니다.

The method of creating object instances is different, but the instance created with "new N.comm()" and the instance created with "N().comm" are the same. The first argument of the N () function is set as the first argument of the new N.comm constructor.

N() 함수의 인자 타입 중에서 string 타입은 N.comm 생성자의 첫 번째 인자(url)로 설정되지 않습니다.

Among the argument types of the N() function, the string type is not set as the first argument (url) of the N.comm constructor.
opts|url object|string N/A

N.comm 함수의 두 번째 인자(opts|url)와 같습니다.

Same as the second argument(opts|url) of the N.comm function.
