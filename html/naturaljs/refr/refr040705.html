선언형 옵션 Declarative Options

N.form의 선언형 옵션은 요소의 data-* 속성에 설정하는 옵션입니다.

The declarative option of N.form is an option set in the data-* attribute of an element.

...
<span id="inputDate" data-format='[ ["date", 8] ]'></span>
<input id="inputDate" type="text" data-validate='[ ["required"], ["integer"] ]' data-format='[ ["trimToEmpty"], ["date", 8, "date"] ]'>
...

선언형 옵션은 JSON 표준 형식을 정확하게 준수해야 합니다(프로퍼티 명은 반드시 큰 따옴표로 감싸야합니다). JSON 표준 포맷을 준수하지 않으면 선언형 옵션이 인식되지 않거나 오류가 발생합니다.

Declarative options must conform exactly to the JSON standard format(Property names must be enclosed in double quotes). If you do not comply with the JSON standard format, the declarative options are not recognized or an error occurs.

선언형 옵션에 대한 설명은 API 문서 안내 메뉴의 선언형 옵션 섹션을 참고하세요.

For a description of the declarative options, see the Declarative Options section of the API Documentation Guide menu.

Name Type Default Description
data-format

String of type array[array[string|number]]

Specifies the format rules. For format rules, see the [Format Rule List] tab in the Formatter menu.
