함수 Methods

Name Arguments Type Return Description
validate N/A N/A array[object[array[object]]]

유효성 검증을 실행 합니다.

Executes the validate.

  • 룰들을 객체 옵션으로 지정 할 경우 : 유효성 검증 된 데이터 목록 객체를 반환 합니다.

    If specifying rules as object options : Returns a validated data list object.

  • data-validate 속성이 정의 된 요소가 포함 된 context 요소를 jquery object 로 지정 할 경우 : 유효성 검증 된 데이터 값을 지정 한 요소에 보여 줍니다.

    If a context element containing an element with data-validate property defined is specified as a jquery object : Shows the validated data values ​​in an element.

룰들을 객체 옵션으로 지정 했을 경우 다음과 같은 폼으로 데이터를 반환 합니다.

If specifying rules as object options, it returns data in the following form :

[{
    "컬럼 프로퍼티 명": [
        {
            "검증 룰": "rule",
            "검증 성공 여부": true|false,
            "검증 메시지": "Message"
        }
    ]
}]
[{
    "Column property name": [
        {
            "Verification rule": "rule",
            "Whether the verification succeeded": true|false,
            "Verification message": "Message"
        }
    ]
}]

검증 메시지는 Config(natural.config.js) 의 N.context.attr("data").validator.message 속성에서 설정 할 수 있습니다.

The validation message can be set in the N.context.attr("data").validator.message property of the Config(natural.config.js).
row number N/A

유효성 검증 할 데이터 목록의 행 index.

Row index of the list of data to be validated.

row 인자가 설정되지 않은 경우 다음과 같이 작동합니다.

If the row argument is not set, it works as follows.

  • 룰들을 객체 옵션으로 지정 할 경우 : 모든 행 데이터를 유효성 검증 합니다.

    If specifying rules as object options : Validates all row data.

  • data-validate 속성이 정의 된 요소가 포함 된 context 요소를 jquery object 로 지정 할 경우 : 첫번째 행 인덱스로 자동 지정 됩니다.

    If a context element containing an element with data-validate property defined is specified as a jquery object : It is automatically specified as the first row index.
