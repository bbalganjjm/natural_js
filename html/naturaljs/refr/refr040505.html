선언형 옵션 Declarative Options

N.tab의 선언형 옵션은 탭(li) 요소의 data-opts 속성에 설정하는 옵션입니다.

The declarative option of N.tab is set in the data-opts attribute of the tab(li) element.

...
<li data-opts='{ "url" : "page.html", "onOpen" : "onOpenFn", "preload" : true, "active" : true }'><a href="#tab01">Tab01</a></li>
...

선언형 옵션은 JSON 표준 형식을 정확하게 준수해야 합니다(프로퍼티 명은 반드시 큰 따옴표로 감싸야합니다). JSON 표준 포맷을 준수하지 않으면 선언형 옵션이 인식되지 않거나 오류가 발생합니다.

Declarative options must conform exactly to the JSON standard format(Property names must be enclosed in double quotes). If you do not comply with the JSON standard format, the declarative options are not recognized or an error occurs.

선언형 옵션에 대한 설명은 API 문서 안내 메뉴의 선언형 옵션 섹션을 참고하세요.

For a description of the declarative options, see the Declarative Options section of the API Documentation Guide menu.

Name Type Default Description
url string undefined

탭 컨텐츠로 생성할 페이지의 URL을 지정합니다.

Specifies the URL of the page to be created as tab content.
active boolean false

true로 설정하면 N.tab이 초기화된 후 해당 탭과 탭 컨텐츠가 기본으로 선택됩니다.

If set to true, N.tab is initialized and the corresponding tab and tab contents are selected by default.

active 옵션은 기본으로 표시할 탭 1개만 true로 설정해야 합니다.

The active option should be set to true for only one tab to be displayed by default
preload boolean false

true로 설정하면 탭 컨텐츠가 최초로 선택되었을 때 페이지를 로딩하지 않고 탭이 초기화될 때 페이지를 미리 로딩합니다.

If set to true, the page is pre-loaded when the tab is initialized without loading the page when the tab content is first selected.

탭을 초기화할 때 탭 컨텐츠의 요소나 Controller object를 참조해야 할 때 사용합니다.

It is used when referencing the element of the tab content or the Controller object when initializing the tab.
onOpen string undefined

탭이 열릴 때마다 불러온 컨텐츠에서 실행될 onOpen 이벤트 핸들러 함수의 이름을 문자열로 지정합니다.

Specifies as a string the name of the onOpen event handler function to be executed on the loaded content whenever the tab is opened.

  • onOpen 이벤트 핸들러 함수는 로딩될 페이지의 Controller object에 지정한 이름으로 정의되어 있어야 합니다.
  • onOpen 이벤트 핸들러 함수는의 첫 번째 인수는 open 함수를 호출할 때 두 번째 인수로 지정한 onOpenData입니다.
  • 탭 컨텐츠가 처음 로딩될 때 N.cont 오브젝트의 init 함수가 실행된 다음 onOpen 함수가 실행됩니다.
  • onActive 이벤트가 정의되어 있으면 onActive 함수가 실행된 다음 onOpen 함수가 실행됩니다.
  • The onOpen event handler function must be defined with the name specified in the Controller object of the page to be loaded.
  • The first argument of the onOpen event handler function is onOpenData specified as the second argument when calling the open function.
  • When the tab content is first loaded, the init function of the N.cont object is executed, then the onOpen function.
  • If the onActive event is defined, the onActive function is executed and then the onOpen function is executed.
disable boolean false

true로 설정하면 지정 한 탭이 비활성화된 상태로 생성됩니다.

If set to true, the specified tab will be created with disabled.
stateless boolean false

true로 설정하면 탭 컨텐츠의 상태를 유지하지 않고 탭을 선택할 때마다 연결된 탭 컨텐츠의 내용을 다시 불러와 초기화합니다.

If set to true, the contents of the linked tab contents are reloaded and initialized each time a tab is selected without maintaining the status of the tab contents.
