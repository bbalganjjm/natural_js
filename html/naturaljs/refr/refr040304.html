If set to false, the datepicker is not displayed when the cursor is focused on the input element.

holiday object[object[string|array]] 
{
    "repeat" : null,
    "once" : null
}

holiday 옵션을 설정하면 Datepicker에 휴일을 표시해 줍니다.

Setting the holiday option displays the holiday in the Datepicker.

아래와 같이 repeat 객체와 once 객체로 휴일들을 설정할 수 있습니다.

Holidays can be set as repeat object and once object as below.

{
    "repeat" : {
        "0619" : "Holiday1",
        "0620" : "Holiday3",
        "0621" : ["Holiday6", "Holiday7"],
        "0622" : ["Holiday9", "Holiday10"]
    },
    "once" : {
        "20200619" : "Holiday2",
        "20200620" : ["Holiday4", "Holiday5"],
        "20200621" : "Holiday8",
        "20200622" : ["Holiday11", "Holiday12"]
    }
}

repeat 객체에는 매년 반복되는 휴일들을 연도를 제외하고 입력합니다. once 객체는 매년 반복되지 않는 휴일들을 연월일 형식으로 입력합니다.

In the repeat object, enter holidays that repeat every year, excluding the year. In the once object, enter holidays that are not repeated every year in the form of year, month, day.

같은 날짜에 휴일이 2개 이상일 경우 Array 타입으로 휴일명을 여러 개 지정하면 됩니다.

If there are two or more holidays on the same day, you can specify multiple holiday names in the Array type.

Config(natural.config.js)의 N.context.attr("ui").datepicker.holiday 프로퍼티에 설정하면 모든 Datepicker에 적용됩니다.

If set in the N.context.attr("ui").datepicker.holiday property of Config(natural.config.js), it applies to all Datepickers.

N.comm("getHolidayList.json").submit(function(data) {
    var once = {};
    N(data).each(function() {
        once[this.holidayDate] = this.holidayName;
    });
    if(N.context.attr("ui").datepicker.holiday === undefined) {
        N.context.attr("ui").datepicker.holiday = {};
    }
    N.context.attr("ui").datepicker.holiday.once = once;
});

휴일로 표시되는 요소에는 "datepicker_holiday__" class 속성 값이 추가됩니다.

"Datepicker_holiday__" class attribute value is added to elements displayed as holidays./p>
