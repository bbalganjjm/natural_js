예제 Examples

⊙ 예제 데이터

⊙Example data


var data = [{
        "key" : "blue",
        "val" : "blue"
    },{
        "key" : "brown",
        "val" : "brown"
    },{
        "key" : "green",
        "val" : "green"
}]

key 옵션과 val 옵션은 Config(natural.config.js)에 다음과 같이 전역 옵션으로 설정하면 N(data).select(".eyeColor").bind();와 같이 간편하게 사용할 수 있습니다.

The key and val options can be conveniently used as N(data).select(".eyeColor").bind(); by setting them as global options in Config(natural.config.js) as follows:


N.context.attr("ui", {
    ...
    "select": {
        "key": "key",
        "val": "val"
    }
    ...
});

다음 예제들은 Config(natural.config.js)에 다음과 같이 설정되어 있다는 가정하에 구동되는 예제입니다. 그렇지 않다면 컴포넌트 초기화 시 key, val 옵션을 설정해야 합니다.

The following examples are based on the assumption that Config(natural.config.js) is configured as follows. Otherwise, you must set the key and val options when initializing the component.

1. select 요소에 데이터 바인딩하기.

1. Binding data to the select element.

1 ~ 4 예제는 context 요소는 다르지만 컴포넌트 초기화 방법은 동일합니다.

In the example 1-4, the context element is different, but the component initialization method is the same.

<select class="eyeColor">
    <option>Select</option>
</select>
<script type="text/javascript">
    N(data).select(".eyeColor").bind();
</script>

1.1. select 요소에 데이터 바인딩하고 기본값으로 "brown" 선택하기.

1.1. Data binding to the select element and selecting "brown" as the default.

<select class="eyeColor">
    <option>Select</option>
</select>
<script type="text/javascript">
    N(data).select(".eyeColor").bind().val("brown");
</script>

2. select[multiple=multiple] 요소에 데이터 바인딩하기.

2. Binding data to the select[multiple=multiple] element.

<select class="eyeColor" multiple="multiple"></select>
<script type="text/javascript">
    N(data).select(".eyeColor").bind();
</script>

2.1. select[multiple=multiple] 요소에 데이터 바인딩하고 기본값으로 "brown"과 "green" 선택하기.

2.1. Data binding to the select[multiple=multiple] element and selecting "brown" and "green" as the default.

<select class="eyeColor" multiple="multiple"></select>
<script type="text/javascript">
    N(data).select(".eyeColor").bind().val(["brown", "green"]);
</script>
