생성자 Constructor

Name Arguments Type Return Description
N.tree N/A N/A N.tree

N.tree 인스턴스를 생성합니다.

Create an N.tree instance.

var tree = N.tree(data, opts|context);
data array[json object] N/A

data 옵션을 설정합니다. 기본 옵션의 data 옵션과 같습니다.

Set data options. Same as the data option of the default options.
opts|context object N/A

컴포넌트의 기본 옵션 객체를 지정합니다.

Specifies the default option object for the component.

jQuery selector 문자열이나 jQuery object를 입력하면 기본 옵션의 context 옵션으로 설정됩니다.

Entering a jQuery selector string or jQuery object sets it as the context option of default options.
N(obj).tree N/A N/A N.tree

N.tree의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

Create an object instance of N.tree with jQuery plugin method.

var tree = N(data).tree(opts|context);

객체 인스턴스를 생성하는 방식은 다르지만 "new N.tree()"로 생성한 인스턴스와 "N().tree"로 생성한 인스턴스는 동일합니다. N() 함수의 첫 번째 인수가 new N.tree 생성자의 첫 번째 인수로 설정됩니다.

The method of creating object instances is different, but the instance created with "new N.tree()" and the instance created with "N().tree" are the same. The first argument of the N () function is set as the first argument of the new N.tree constructor.
opts|context object N/A

N.tree 함수의 두 번째 인수(opts)와 같습니다.

Same as the second argument(opts) of the N.tree function.
