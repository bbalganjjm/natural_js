함수 Methods

Name Arguments Type Return Description
context N/A N/A jQuery object

context 요소를 반환합니다.

Returns the context element.
selector string N/A

jQuery selector 구문을 입력하면 context 요소에서 지정한 요소를 찾아서 반환합니다.

If you input jQuery selector syntax, the element specified in the context element is found and returned.
add N/A N/A N.docs

새로운 탭 컨텐츠를 추가합니다.

Adds new tab content.
docId string N/A

document 아이디를 입력합니다.

Inputs the document id.

동일한 아이디를 가진 탭 컨텐츠가 이미 있으면 컨텐츠가 활성화됩니다.

If there is already tab content with the same id, the content is activated.
docNm string N/A

document 이름을 입력합니다.

Inputs the document name.
docOpts object N/A

추가되는 페이지에만 적용되는 옵션을 입력합니다.

Input options that apply only to the page being added.

var docOpts = {

    docId : docId, // document 아이디
    docNm : docNm, // document 명
    onBeforeLoad : function(docId, target) {
        // 추가되는 document 에만 적용되는 onBeforeLoad 이벤트.

        // docId : document id
        // target : 로드된 컨텐츠를 넣어 줄 요소

        var doc = this.doc(docId); // document 정보 가져오기
	    var cont = this.cont(docId); // document의 Controller object 가져오기
	    var view = cont.view;
	    var request = cont.request;
    },
    onLoad : function(docId) {
        // 추가되는 document 에만 적용되는 onLoad 이벤트.
    },
    onBeforeActive : function(docId) {
        // 추가되는 document 에만 적용되는 onBeforeActive 이벤트.
    },
    onActive : function(docId) {
        // 추가되는 document 에만 적용되는 onActive 이벤트.
    },
    onBeforeInactive : function(docId) {
        // 추가되는 document 에만 적용되는 onBeforeInactive 이벤트.
    },
    onInactive : function(docId) {
        // 추가되는 document 에만 적용되는 onInactive 이벤트.
    },
    onBeforeRemoveState : function(docId) {
        // 추가되는 document 에만 적용되는 onBeforeRemoveState 이벤트.
    },
    onRemoveState : function(docId) {
        // 추가되는 document 에만 적용되는 onRemoveState 이벤트.
    },
    onBeforeRemove : function(docId) {
        // 추가되는 document 에만 적용되는 onBeforeRemove 이벤트.
    },
    onRemove : function(docId) {
        // 추가되는 document 에만 적용되는 onRemove 이벤트.
    },
    stateless : false // 탭 컨텐츠의 상태 유지 여부
}

var docOpts = {
    url : "url", // document URL
    urlSync : true, // The response will be blocked if location.href when requesting to the server and location.href when receiving a response from the server are different.
    docId : docId, // document id
    docNm : docNm, // document name
    onBeforeLoad : function(docId, target) {
        // onBeforeLoad event that applies only to the document being added.

        // docId : document id
        // target : Element to put loaded content

	    var doc = this.doc(docId); // get document info
	    var cont = this.cont(docId); // get document's Controller object
	    var view = cont.view;
	    var request = cont.request;
    },
    onLoad : function(docId) {
        // onLoad event that applies only to the document being added.
    },
    onBeforeActive : function(docId) {
        // onBeforeActive event that applies only to the document being added.
    },
    onActive : function(docId) {
        // onActive event that applies only to the document being added.
    },
    onBeforeInactive : function(docId) {
        // onBeforeInactive event that applies only to the document being added.
    },
    onInactive : function(docId) {
        // onInactive event that applies only to the document being added.
    },
    onBeforeRemoveState : function(docId) {
        // onBeforeRemoveState event that applies only to the document being added.
    },
    onRemoveState : function(docId) {
        // onRemoveState event that applies only to the document being added.
    },
    onBeforeRemove : function(docId) {
        // onBeforeRemove event that applies only to the document being added.
    },
    onRemove : function(docId) {
        // onRemove event that applies only to the document being added.
    },
    stateless : false // Whether the tab content is maintained
}

여기에 설정된 이벤트들은 전역 이벤트가 실행된 다음 실행됩니다.

Events set here are executed after the global event.
active N/A N/A N.docs

지정한 탭 컨텐츠를 활성화합니다.

Activates the specified tab content.
docId string N/A

document 아이디를 입력합니다.

Inputs the document id.
removeState N/A N/A N.docs

지정한 탭 컨텐츠의 상태를 제거합니다.

Removes the state of the specified tab content.
docId string N/A

document 아이디를 입력합니다.

Inputs the document id.
remove N/A N/A N.docs

지정한 탭 컨텐츠를 제거합니다.

Removes the specified tab content.
docId string N/A

document 아이디를 입력합니다.

Inputs the document id.
unconditional boolean N/A

true를 입력하면 페이지에 수정된 데이터가 있더라도 무조건 제거합니다.

If you input true, even if there is modified data on the page, it will be removed unconditionally.
doc N/A N/A object|array

지정한 탭 컨텐츠의 정보(docOpts)를 가져옵니다.

Get information(docOpts) of the specified tab content.
docId string N/A

document 아이디를 입력합니다.

Inputs the document id.

입력하지 않으면 모든 탭 컨텐츠의 정보를 배열로 반환합니다.

If not input, information of all tab contents is returned as an array.
cont N/A N/A object

지정한 탭 컨텐츠의 Controller object를 가져옵니다.

Gets the Controller object of the specified tab content.
docId string N/A

document 아이디를 입력합니다.

Inputs the document id.
reload N/A N/A N.docs

지정한 탭 컨텐츠를 다시 불러옵니다.

Reloads the specified tab content.

reload 함수를 실행하기 전에 request.attr 함수로 값을 지정하지 않았다면 리로드 전의 request 객체의 값은 리로드 해도 계속 유지됩니다.

If the value is not specified by request.attr before executing the reload function, the value of the request object before the reload is retained even after reloading.
docId string N/A

document 아이디를 입력합니다.

Inputs the document id.
callback function N/A

리로드가 완료되면 실행될 콜백을 지정합니다. 필요 없으면 지정하지 않아도 됩니다.

Callback to be executed when reload is completed. If you don't need it, you don't need to specify it.

콜백 함수의 인자로 html과 Communicator.request 객체가 입력됩니다.

The html and Communicator.request objects are input to the arguments of callback function.

docsInstance.reload("docId", function(html, request) {
    N.log(html, request);
});
