함수 Methods

Name Arguments Type Return Description
data N/A N/A array[json object]|jQuery object[array[json object]]

컴포넌트에 바인드 되어 있는 가장 최근 상태의 데이터를 반환합니다.

Returns the data of the most recent state bound to the component.
selFlag boolean N/A

반환되는 데이터의 타입을 결정하는 구분값입니다.

This is a division value that determines the type of data returned.

selFlag 인자 값에 따라 다음과 같은 데이터를 반환합니다.

The following data is returned according to the value of the selFlag argument.

  • undefined(selFlag 옵션이 설정되지 않은 경우) : array[json object] 타입의 데이터를 반환합니다.
  • undefined(When the selFlag option is not set) : Returns data of type array [json object].
  • true : 현재 선택 된 row 데이터만 추출하여 array 타입으로 반환합니다.
  • true : Only the currently selected row data is extracted and returned as an array type.

  • false : 컴포넌트에 바인딩 되어 있는 jQuery object[array[json object]] 타입의 원래 유형의 데이터를 반환합니다.

    false : Returns the data of the original type of the jQuery object[array[json object]] type bound to the component.

    data 메서드로 가져온 데이터를 다른 데이터 관련 컴포넌트에 바인딩할 때는 반드시 "false" 로 설정하여 원래 유형의 데이터를 바인딩해야 양방향 데이터 바인딩이 활성 화 됩니다.

    When binding data imported by the data method to other data-related components, you must set "false" to bind the original type of data to enable two way databinding.
context N/A N/A jQuery object

context 요소를 반환합니다.

Returns the context element.
selector string N/A

jQuery selector 구문을 입력하면 context 요소에서 지정한 요소를 찾아서 반환합니다.

If you input jQuery selector syntax, the element specified in the context element is found and returned.
bind N/A N/A N.select

context 옵션으로 지정한 요소에 데이터를 바인딩 합니다.

Binds data to the element specified by context option.

context 요소가 체크박스 일 경우 체크된 상태라면 데이터가 바인딩 될 때 생성되는 요소들도 체크된 상태로 생성됩니다.

If the context element is a checkbox, if it's state is checked, the elements created when the data is bound are also created as checked.

radio 나 checkbox input 요소는 데이터가 바인드 되면 context 로 설정 한 요소에는 id 속성과 name 속성이 있고 추가로 생성되는 옵션 요소에는 name 속성만 추가 됩니다. 생성된 체크박스나 라디오에 이벤트를 바인드할때 id selector 로 요소를 선택하면 첫 번째 옵션 요소에만 이벤트가 적용되니 name 속성으로 요소를 선택 바랍니다.

When the data is bound to the radio or checkbox input element, the element set as context has the id attribute and the name attribute, and only the name attribute is added to the additionally created option element. When selecting an element with the id selector when binding an event to the created checkbox or radio, the event is applied only to the first option element, so select the element with the name attribute. 

$("input[name='name']").on("click", function(e) { ... });
data array[json object] N/A

바인딩할 데이터를 지정 합니다.

Specifies the data to be bound.
val N/A N/A string|array

선택된 요소의 값을 얻거나 선택 합니다.

Gets or selects the value of the selected option element.
val string N/A

선택할 값.

Value of option to select.
index N/A N/A number|array[number]

선택된 요소의 index 를 얻거나 선택 합니다.

Gets or selects the index of the selected option element.
index number|array[number] N/A

선택할 index.

Index of option to select.
remove val number N.select

val 인자로 지정한 값과 같은 옵션 요소와 데이터 객체를 제거 합니다.

Removes optional elements and data objects equal to the value specified by the val argument.
reset N/A N/A N.select

선택을 재설정합니다.

Resets selection.
selFlag boolean N/A

select 요소의 경우 true 로 설정하면 아무것 선택하지 않고 false로 설정하면 기본(첫 번째) 옵션 요소가 선택 됩니다.

For select elements, if set to true, nothing is selected; if set to false, the default (first) option element is selected.
