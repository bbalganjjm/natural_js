N.browser 객체의 함수 Function of N.browser Object

N.browser 은 browser 관련 유틸리티 함수들을 모아놓은 객체입니다.

N.gc is a collection object of utility functions related to browser.

"N.browser.{함수명}(arg[0...N])"와 같이 사용할 수 있습니다.

You can use like "N.browser.{function name}(arg[0...N])"

Name Arguments Type Return Description
cookie N/A N/A undefined

브라우저 쿠키를 생성합니다.

Creates the browser cookie.
name string N/A

쿠키명을 입력합니다.

Inputs the cookie name.
value string N/A

쿠키값을 입력합니다.

Inputs the cookie value.
expiredays number N/A

쿠키의 만료일을 입력합니다.

Inputs the expiration date of the cookie.
domain string N/A

쿠키가 생성될 도메인을 입력합니다.

Inputs the domain where the cookie will be created.
removeCookie N/A N/A undefined

생성된 쿠키를 제거합니다.

Removes created cookie.
name string N/A

쿠키명을 입력합니다.

Inputs the cookie name.
domain string N/A

제거할 쿠키의 도메인을 입력합니다.

Inputs the domain of the cookie to be removed.
msieVersion N/A N/A number

접속한 MSIE(Microsoft Internet Explorer)의 버전을 가져옵니다. MSIE가 아닐 경우 0을 반환합니다.

Gets the version of the MSIE(Microsoft Internet Explorer) you are connected to. Returns 0 if not MSIE.
is N/A N/A boolean

접속한 브라우저나 모바일 OS가 입력한 name 인자 값과 일치하면 true를 반환합니다.

Returns true if the connected browser or mobile OS matches the name argument value input.
name string N/A

브라우저명이나 모바일 OS명을 입력합니다.

Inputs the browser name or mobile OS name.

  • "opera" : Opera Browser.
  • "firefox" : Mozilla Firefox Browser.
  • "safari" : Apple Safari Browser.
  • "chrome" : Google Chrome Browser.
  • "ie" : Microsoft Internet Explorer Browser.
  • "android" : Google Android OS
  • "ios" : Apple iOS
contextPath N/A N/A string

브라우저 URL에서 컨텍스트 경로를 반환합니다.

Returns the context path from the browser URL.
scrollbarWidth N/A N/A number

브라우저 스크롤바의 넓이를 반환합니다.

Returns the width of the browser's scroll bar.
