함수 Methods

Name Arguments Type Return Description
attr N/A N/A set:Controller.request|get:anything

로드할 페이지로 전달할 데이터를 지정하거나 전달된 데이터를 가져옵니다.

Specifies the data to be passed to the page to be loaded or get the delivered data.

attr 메서드의 인자의 개수가 2개이면 set으로 작동하고 1개이면 get으로 작동합니다.

If the number of arguments of the attr method is 2, it works as set, and if 1, it works as get.

데이터 전달 하기 :

Passing data :

N(".view").cont({
    init : function(view, request) {
        N("#section").comm("page.html")
            .request.attr("data1", { data : ["1", "2"] })
            .request.attr("data2", ["3", "4"])
                .submit();
    }
});

로딩된 페이지에서 데이터 가져오기 :

Get data from the loaded page :

N(".view").cont({
    init : function(view, request) {
        var data1 = request.attr("data1"); // { data : ["1", "2"] }
        var data2 = request.attr("data2"); // ["3", "4"]
    }
});
name string N/A

데이터명을 지정합니다.

Specifies the data name.
value anything N/A

데이터 값을 지정합니다.

Specifies the data value.
param N/A N/A string|object

브라우저의 URL에서 GET 파라미터 값을 추출합니다.

Extracts the GET parameter value from the browser URL.
name string N/A

파라미터의 key를 지정합니다. 지정하지 않으면 GET 파라미터 전체를 object 타입으로 반환합니다.

Specifies the key of the parameter. If not specified, the entire GET parameter is returned as an object type.
get N/A N/A Value object

Communicator.request 인스턴스에 설정된 기본 옵션 정보를 가져옵니다.

Get the default option information set on the Communicator.request instance.

기본 옵션 외에도 "referrer"(submit 메서드를 호출할 때의 브라우저 URL) 값을 추가로 가져올 수 있습니다.

In addition to the default options, you can get "referrer"(Browser URL when submit method is called) value.
key string N/A

기본 옵션명을 지정합니다. 지정하지 않으면 기본 옵션 객체가 반환됩니다.

Specifies the default option name. If not specified, default option object is returned
reload N/A N/A Controller object

Communicator로 로드 한 블록 페이지를 다시 로딩 합니다.

Reloads the block page loaded by Communicator.

reload 메서드를 호출하기 전에 attr 메서드로 값을 지정하지 않았다면 reload 전의 request 객체의 값은 reload해도 유지됩니다.

If the value is not specified with the attr method before calling the reload method, the value of the request object before reload is retained even when reloading.

attr 메서드로 reload 되는 페이지의 Communicator.request 데이터를 다시 지정할 수 있습니다.

The request data of the page reloaded by the attr method can be specified again.

request.attr("param", { param : 1 });
request.reload();

reload 함수는 메서드 체이닝을 지원하지 않습니다.

The reload function does not support method chaining.

request.attr("param", { param : 1 }).reload(); // 이렇게 사용할 수 없습니다.
request.attr("param", { param : 1 }).reload(); // Can not be used like this.
callback string N/A

리로드가 완료되면 실행되는 콜백을 지정합니다. 필요 없으면 지정하지 않아도 됩니다.

Specifies the callback to be executed when reloading is complete. You do not need to specify it if you do not need it.

request.reload(function(html, request) {
    N.log(html, request);
});
