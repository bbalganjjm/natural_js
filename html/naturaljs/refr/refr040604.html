기본옵션 Default Options

Name Type Default Required Description
data array[json object] undefined O

N.select에 바인딩할 데이터를 지정 합니다.

Specifies the data to bind to N.select.
context jquery object|jquery selector string null O

N.select를 적용할 context 요소를 지정 합니다.

Specifies the context element to apply N.select.

N.select의 context 요소는 반드시 select 나 input[type=checkbox], input[type=radio] 태그로 작성 해야 합니다.

The context element of N.select must be written with select or input[type=checkbox] or input[type=radio].

<select class="select-context">
    <option value="">Select</option>
</select>
<select class="select-context" multiple="multiple"></select>
<input class="select-context" type="radio">
<input class="select-context" type="checkbox">
key string null O

선택 요소의 name 속성에 바인드 될 데이터의 프로퍼티명을 지정 합니다.

Specifies the property name of the data to be bound to the name attribute of the selection element.
val string null O

선택 요소의 value 속성에 바인드 될 데이터의 프로퍼티명을 지정 합니다.

Specifies the property name of the data to be bound to the value attribute of the selection element.
append boolean true X

false로 설정하면 select 요소에서 기본 option 들을 비우고 나서 데이터를 바인드 합니다.

If set to false, empty the default options in the select element and then bind the data.
direction string h X

context 가 input[type=checkbox] 이나 input[type=radio] 일 경우 선택요소의 배치 방향을 설정 합니다.

If the context is input [type = checkbox] or input [type = radio], sets the placement direction of the selected element.

  • h : 가로
  • h : horizontal
  • v : 세로
  • v : vertical
