Name Arguments Type Return Description
data N/A N/A array[json object]|jQuery object[array[json object]]

컴포넌트에 바인딩되어 있는 가장 최근 상태의 데이터를 반환합니다.

Returns the data of the most recent state bound to the component.
selFlag boolean N/A

반환되는 데이터의 타입을 결정하는 구분 값입니다.

This is a division value that determines the type of data returned.

selFlag 인수 값에 따라 다음과 같은 데이터를 반환합니다.

The following data is returned according to the value of the selFlag argument.

  • undefined(selFlag 옵션이 설정되지 않은 경우) : array[json object] 타입의 데이터를 반환합니다.
  • undefined(When the selFlag option is not set) : Returns data of type array [json object].
  • true : 폼에 바인딩되어 있는 데이터만 추출하여 반환합니다.
  • true : It extracts and returns only the data bound to the form.

  • false : 컴포넌트에 바인딩되어 있는 jQuery object[array[json object]] 타입의 원래 유형의 데이터를 반환합니다.

    false : Returns the data of the original type of the jQuery object[array[json object]] type bound to the component.

    data 메서드로 가져온 데이터를 다른 데이터 관련 컴포넌트에 바인딩할 때는 반드시 "false"로 설정하여 원래 유형의 데이터를 바인딩해야 양방향 데이터 바인딩이 활성화됩니다.

    When binding data imported by the data method to other data-related components, you must set "false" to bind the original type of data to enable two way databinding.

 string N/A

data 메서드의 두 번째 인수부터 n 번째 인수까지 데이터의 프로퍼티명을 인수로 지정하면 지정한 프로퍼티 값만 추출한 객체를 반환합니다.

If you specify the property name of the data from the second argument to the nth argument of the data method as an argument, the object that extracts only the specified property value is returned.

var formInst = N([]).form(".context")
    .bind(0, [{ col01 : "", col02 : "", col03 : "", col04 : "", col05 : "", col06 : "" }]);
formInst.data(true, "col01", "col02", "col03");
    // [{ col01 : "", col02 : "", col03 : "" }]

첫 번째 인수를 true로 설정했을 때만 작동됩니다.

It works only when the first argument is set to true.
row N/A N/A number

바인딩되어 있는 데이터 오브젝트의 index를 반환합니다.

Returns the index of the bound data object.
context N/A N/A jQuery object

context 요소를 반환합니다.

Returns the context element.
selector string N/A

jQuery selector 구문을 입력하면 context 요소에서 지정한 요소를 찾아서 반환합니다.

If you input jQuery selector syntax, the element specified in the context element is found and returned.
bind N/A N/A N.form

context 옵션으로 지정한 요소 안에서 id 속성 값을 갖고 있는 요소들에 데이터를 바인딩합니다.

Binds data to elements that have id attribute value in the element specified by context option.

data 만 바꿔서 다시 바인딩하면 N.form이 제대로 동작하지 않을 수 있습니다. 데이터가 바인딩된 context 요소에 다시 바인딩할 때는 unbind 메서드를 먼저 실행한 다음 bind 메서드를 실행해야 합니다.

If you change data only and bind again, N.form may not work properly. When rebinding to the context element to which the data is bound, the unbind method must be executed first, then the bind method.
row number N/A

지정한 data(array[json object])에서 바인딩할 데이터 오브젝트의 행 index를 설정합니다.

Set the row index of the data object to be bound in the specified data(array[json object]).
data array[json object] N/A

바인딩할 데이터를 지정합니다.

Specifies the data to be bound.

arguments[2],

 string N/A

bind 메서드의 세 번째 인수부터 n 번째 인수까지 데이터의 프로퍼티명을 지정하면 프로퍼티명과 id 속성 값이 일치하는 요소만 데이터가 바인딩됩니다.

If you specify the property name of the data from the third argument to the nth argument of the bind method, only the element whose property name matches the value of the id attribute is bound data.

// 바인딩할 데이터
var data = [{ col01 : "", col02 : "", col03 : "", col04 : "", col05 : "", col06 : "" }]
// "col01", "col02", "col03" id 속성 값을 갖고 있는 요소에만 바인딩.
formInstance.bind(0, data, "col01", "col02", "col03");
// Data to bind
var data = [{ col01 : "", col02 : "", col03 : "", col04 : "", col05 : "", col06 : "" }]
// "col01", "col02", "col03" Only bound to elements that have an id attribute value.
formInstance.bind(0, data, "col01", "col02", "col03");
// 바인딩되어 있는 데이터를 직접 수정한 후 "col01", "col02", "col03" id 속성 값을 갖고 있는 요소에만 바인딩.
var data = formInstance.data()[formInstance.row()]
data.col01 = "abc";
data.col02 = "123";
data.col03 = "가나다";
formInstance.bind(0, data, "col01", "col02", "col03");
// After directly modifying the bound data, binds to only elements that have the "col01", "col02", and "col03" id attribute values.
var data = formInstance.data()[formInstance.row()]
data.col01 = "abc";
data.col02 = "123";
data.col03 = "가나다";
formInstance.bind(0, data, "col01", "col02", "col03");
unbind N/A N/A N.form

context로 지정한 요소에서 N.form 관련 이벤트나 상태, 스타일등을 제거하여 bind 하기 전 상태로 되돌려 줍니다.

Removes N.form-related events, states, styles, etc. from the element specified as context and revert to the state of before binding.

폼에 바인딩되어있는 data는 제거되지 않고 선택된 row 값(this.options.row)도 그대로 유지됩니다.

The data bound to the form is not removed and the selected row value(this.options.row) is also retained.
add N/A N/A N.form

새로운 행 데이터를 추가합니다.

Adds new row data.

행 데이터 생성 시 context 요소 안에 있는 입력 요소들의 id 속성명과 값으로 데이터 객체를 생성합니다.

When creating row data, a data object is created with the id attribute name and value of the input elements in the context element.

N.form 초기화 후 add 메서드를 실행하면 새로 생성된 데이터로 바인딩됩니다.

If the add method is executed after N.form initialization, it is bound by the newly created data.
data object|number N/A

data 객체를 지정하면 생성된 데이터와 지정 한 data가 병합되어 바인딩됩니다.

When a data object is specified, the added data and the specified data are merged and then bound.

data 인수가 number 타입이면 row(arguments[1]) 인수로 지정됩니다.

If the data argument is of type number, it is specified as a row (arguments [1]) argument.
row number N/A

새로운 데이터가 추가될 행 인덱스를 row 인수로 지정하면 지정한 행 인덱스 바로 앞에 행 데이터가 추가됩니다.

If the row index to which new data is to be added is specified as the row argument, the row data is added right in front of the specified row index.
revert N/A N/A N.form

최초 바인딩된 초기 데이터나 add 시 생성된 초기 데이터로 되돌립니다.

Reverts to the initial bound data or initial data created when add.
validate N/A N/A N.form

추가 / 수정된 전체 데이터에 대한 유효성 검증 결과를 반환합니다. 유효성 검증 성공 시 true를 반환하고 실패 시 false가 반환되면서 해당 입력 요소 옆에 툴팁으로 실패 메시지를 표시합니다.

Returns the result of validation on all added / modified data. If validation succeeds, it returns true, and on failure, false is returned, and a tooltip is displayed next to the input element.
val N/A N/A string|number|boolean|N.form

컴포넌트에 바인딩되어 있는 데이터의 프로퍼티 값을 얻어오거나 설정합니다.

Gets or sets the property value of data bound to the component.

val 인수가 설정되면 지정된 값을 바인딩된 데이터에 세팅하거나 추가하고 val 인수가 설정되지 않으면 지정된 값을 얻어옵니다.

If the val argument is set, the specified value is set or added to the bound data. If the val argument is not set, the specified value is get.
key string N/A

바인딩된 데이터 객체의 프로퍼티명

Property name of the bound data object
val string|number|boolean|null N/A

설정 값.

Setting value.

데이터 값이 바인딩된 요소가 있으면 null 은 빈 문자열로, number 타입은 숫자 문자열로, boolean 은 "true" 나 "false" 문자열로 변환되어 설정됩니다. 바인딩된 요소가 없으면 설정한 값의 원래 타입으로 저장됩니다.

If there is an element to which the data value is bound, null is converted to an empty string, number type is a numeric string, and boolean is converted to a "true" or "false" string and set. If there is no bound element, it is saved as the original type of the setting value.
