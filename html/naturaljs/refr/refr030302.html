함수 Methods

Name Arguments Type Return Description
N.data.filter N/A N/A array[json object]

지정한 조건과 일치하는 데이터를 추출합니다.

Extract data matching specified conditions.

var filteredData = N.data.filter(data, condition);
data array[json object] N/A

필터링할 data

Data to be filtered
condition function|string N/A

필터링 조건을 지정합니다.

Specifies filtering conditions.

인자로 함수를 지정하면 함수에서 true를 반환한 행들만 필터링합니다.

If a function is specified as an argument, only rows that return true in the function are filtered.

var fData = N.data.filter([
    { name : "John", age : 18 },
    { name : "Mike", age : 16 },
    { name : "Mike", age : 14 }
], function(item) {
    return item.name === "Mike" && item.age === 16;
});

console.log(fData); // [{ name : "Mike", age : 16 }]

인자로 조건 문자열을 지정하면 조건에 맞는 행들 만 필터링합니다.

If a condition string is specified as an argument, only rows matching the condition are filtered.

var fData = N.data.filter([
    { name : "John", age : 18 },
    { name : "Mike", age : 16 },
    { name : "Mike", age : 14 }
], 'name === "Mike"');

console.log(fData); // [{ name : "Mike", age : 16 }, { name : "Mike", age : 14 }]

문자열로 조건을 지정하는것 보다 함수로 조건을 처리하는 방식이 더 빠릅니다.

It is faster to handle conditions with functions than to specify conditions with strings.

문자열로 조건을 지정하는 방식은 and(&&) 나 or(||) 연산식을 지원하지 않고 단일 조건식만 지원합니다.

The method of specifying a condition as a string does not support the expressions of and(&&) and or(||), but only a single conditional expression.
N.data.sort N/A N/A array[json object]

지정한 "key" 인자 값을 기준으로 데이터를 정렬합니다.

Sorts data based on the specified "key" argument value.

var sortedData = N.data.sort(data, key, reverse);
data array[json object] N/A

정렬할 data

Data to be sorted
key string N/A

정렬 기준이 될 object의 프로퍼티 명

Property name of the object to be sorted by
reverse boolean N/A

true로 설정하면 내림차순으로 정렬됩니다.

If set to true, it is sorted in descending order.
N(data).datafilter N/A N/A jQuery object[json object]

N.data.filter 함수를 jQuery 플러그인 방식으로 실행합니다.

Executes the N.data.filter function using the jQuery plugin method.

var filteredData = N(data).datafilter(condition);

N() 함수의 첫 번째 인자가 new N.data.filter 생성자의 첫 번째 인자로 설정됩니다.

The first argument of the N () function is set as the first argument of the new N.data.filter constructor.
condition function|string N/A

N.data.filter의 condition 인자와 같습니다.

Same as the condition argument of N.data.filter.

N.data.sort 함수를 jQuery 플러그인 방식으로 실행합니다.

Executes the N.data.sort function using the jQuery plugin method.

The first argument of the N () function is set as the first argument of the new N.data.sort constructor.
key string N/A

N.data.sort의 key 인자와 같습니다.

reverse boolean N/A

N.data.sort의 reverse 인자와 같습니다.
