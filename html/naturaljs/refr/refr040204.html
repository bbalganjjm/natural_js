기본 옵션 Default Options

Name Type Default Required Description
context jQuery object|jquery selector string null O

N.button을 적용할 context 요소를 지정합니다.

Specifies the context element to apply N.button.

N.button의 context 요소는 반드시 a 나 button, input[type=button] 태그로 작성해야 합니다.

The context element of N.button must be written with a or button, input[type=button] tag.

N.button은 한 번에 여러 요소를 선택해서 context로 지정할 수 있습니다.

N.button can select multiple elements at once and set it as context.

<a class="button-context" data-opts='{ "size" : "big" }'>Button</a>
<input class="button-context" type="button" value="Button" data-opts='{ "color": "primary" }'>
<button class="button-context" data-opts='{ "color": "primary", "size" : "large" }'>Button</button>
size string none X

버튼의 크기를 설정합니다.

Set the size of the button.

크기는 "none", "smaller", "small", "medium", "large", "big" 중 하나를 지정할 수 있습니다.

The size can be one of "none", "smaller", "small", "medium", "large" and "big".
color string none X

버튼의 색상을 설정합니다.

Set the color of the button.

색상은 "none", "primary", "primary_container", "secondary", "secondary_container", "tertiary", "tertiary_container" 중 하나를 지정할 수 있습니다.

The color can be one of "none", "primary", "primary_container", "secondary", "secondary_container", "tertiary" and "tertiary_container".

버튼 색상의 Naming 과 기본값은 Material Design 3 의 Color roles 를 기반으로 합니다.

The naming and default value of the button color is based on the Color token of Material Design 3.

버튼의 스타일을 변경하려면 natural.ui.css 파일의 "btn_" 로 시작하는 클래스를 수정하면 됩니다.

To change the style of the button, simply edit the class starting with "btn_" in the natural.ui.css file.

a.btn_{color}__,input[type='button'].btn_{color}__,button.btn_{color}__ {
    color: var(--md-sys-color-on-tertiary);
    background-color: var(--md-sys-color-tertiary);
    border: 1px solid var(--md-sys-color-tertiary);
}
type string none X

버튼의 배경을 채울지 외곽선만 표시할지 설정 합니다.

Set whether to fill the background of the button or display only the outline.

타입은 "none", "filled", "outlined", "elevated" 중 하나를 지정할 수 있습니다.

The type can be one of "none", "filled", "outlined" and "elevated".

"filled", "outlined" 의 색상은 color 옵션으로 정의한 색상으로 지정 됩니다. color 옵션이 "none" 이면 버튼의 배경이나 외곽선이 표시되지 않습니다.

The color of "filled" and "outlined" is specified by the color defined by the color option. If the color option is "none", the button's background or outline will not be displayed.
disable boolean false X

true로 설정하면 버튼이 비활성화된 상태로 생성됩니다.

If set to true, the button is created in a disabled state.
onBeforeCreate function null X

버튼의 옵션이 적용되기 전에 실행되는 이벤트핸들러를 정의 합니다.

Defines an event handler that is fired before the button's options are applied.

외부 버튼 라이브러리를 사용할때 버튼요소의 외부나 내부에 필요한 HTML 요소를 추가하거나 다른 용도로 HTML 요소의 편집이 필요할 때 활용 할 수 있습니다.

When using an external button library, you can add necessary HTML elements to the outside or inside of a button element, or use it when you need to edit HTML elements for other purposes.

onBeforeCreate : function(context, opts) {
    // this : N.button 인스턴스
    // context : context 요소
    // opts : 버튼을 생성할 때 지정된 옵션
}
onBeforeCreate : function(context, opts) {
    // this : N.button instance
    // context : context element
    // opts : options that specified when creating a button
}
onCreate function null X

버튼의 옵션이 적용된 후에 실행되는 이벤트핸들러를 정의 합니다.

Defines an event handler that is fired after the button's options are applied.

외부 버튼 라이브러리를 사용할때 버튼요소에 효과 이벤트를 정의하거나 다른 용도로 버튼 처리가 필요할 때 활용 할 수 있습니다.

When using an external button library, you can define effect events on button elements or use it when button handling is needed for other purposes.

onCreate : function(context, opts) {
    // this : N.button 인스턴스
    // context : context 요소
    // opts : 버튼을 생성할 때 지정된 옵션
}
onCreate : function(context, opts) {
    // this : N.button instance
    // context : context element
    // opts : options that specified when creating a button
}
