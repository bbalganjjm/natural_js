함수 Methods

Name Arguments Type Return Description
context N/A N/A jQuery object

context 요소를 반환합니다.

Returns the context element.
selector string N/A

jQuery selector 구문을 입력하면 context 요소에서 지정한 요소를 찾아서 반환합니다.

If you input jQuery selector syntax, the element specified in the context element is found and returned.
open N/A N/A N.popup

팝업을 오픈합니다.

Opens the popup.
onOpenData object N/A

onOpen 이벤트 옵션으로 지정 한 onOpen 함수의 첫 번째 인자로 onOpenData를 전달합니다.

onOpenData is passed as the first argument of the onOpen function specified with the onOpen event option.

자세한 내용은 기본 옵션 탭의 onOpen 옵션을 참고 바랍니다.

For more information, see the onOpen option in the [Default Options] tab.
close N/A N/A N.popup

팝업을 닫거나 제거합니다.

Closes or removes the popup.
onCloseData object N/A

onClose 이벤트 옵션으로 지정 한 함수의 첫 번째 인자로 onCloseData를 전달합니다.

onCloseData is passed as the first argument of the function specified by the onClose event option.

자세한 내용은 기본 옵션 탭의 onClose 옵션을 참고 바랍니다.

For more information, see the onClose option in the [Default Options] tab.
remove N/A N/A N.popup

N.popup과 관련된 모든 요소를 제거합니다.

Remove all elements related to N.popup.
