Controller object의 상수 Constant for the Controller object

Controller object 에는 다음과 같은 상수 값들이 정의되어 있습니다. The following constant values are defined in the Controller object.
Name Type Description
view jquery object

view 요소입니다.

view element.

init 함수의 첫번째 인자 값과 동일 합니다.

It is the same as the value of the first argument of the init function.
request Communicator.request

Communicator.request 객체 인스턴스입니다.

Communicator.request object instance.

init 함수의 두 번째 인자 값과 동일 합니다.

It is the same as the value of the second argument of the init function.
caller N.popup instance|N.tab instance

N.popup 이나 N.tab 컴포넌트로 호출 된 페이지 일 경우 이 페이지를 로드 한 N.popup 이나 N.tab 인스턴스가 정의되어 있습니다.

In the case of a page called with an N.popup or N.tab component, the N.popup or N.tab instance that loaded this page is defined.
opener N.cont's object

N.popup 이나 N.tab 컴포넌트로 호출 된 페이지 일 경우 이 페이지를 로드 한 N.popup 이나 N.tab 인스턴스가 포함된 Controller object(부모 페이지의 Controller object)가 정의되어 있습니다.

In the case of a page called with an N.popup or N.tab component, a Controller object (Controller object of the parent page) containing the N.popup or N.tab instance that loaded this page is defined.

opener 속성은 N.popup 이나 N.tab 컴포넌트 인스턴스를 생성할 때 opener 옵션으로 해당 페이지의 Controller object를 지정 해 줘야 합니다.

When creating an N.popup or N.tab component instance, the opener property must specify the controller object of the corresponding page with the opener option.
