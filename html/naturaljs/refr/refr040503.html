N.tab 인스턴스를 생성합니다.

Create an N.tab instance.

var tab = N.tab(context, opts);
context jQuery object|jquery selector string N/A

context 옵션을 설정합니다. 기본 옵션의 context 옵션과 같습니다.

Set context options. Same as the context option of the default options.
opts object N/A

컴포넌트의 기본 옵션 객체를 지정합니다.

Specifies the default option object for the component.
N(obj).tab N/A N/A N.tab

N.tab의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

Create an object instance of N.tab with jQuery plugin method.

var tab = N(context).tab(opts);

객체 인스턴스를 생성하는 방식은 다르지만 "new N.tab()"로 생성한 인스턴스와 "N().tab"로 생성한 인스턴스는 동일합니다. N() 함수의 첫 번째 인자가 new N.tab 생성자의 첫 번째 인자로 설정됩니다.

The method of creating object instances is different, but the instance created with "new N.tab()" and the instance created with "N().tab" are the same. The first argument of the N () function is set as the first argument of the new N.tab constructor.
opts object N/A

N.tab 함수의 두 번째 인자(opts)와 같습니다.

Same as the second argument(opts) of the N.tab function.
