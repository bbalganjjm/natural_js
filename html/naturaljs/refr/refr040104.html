Name Type Default Required Description
context window|jQuery object|jquery selector string null O

N.alert의 메시지 대화상자가 표시될 영역을 지정합니다.

Specify the area where the N.alert message dialog will be displayed.

modal 옵션이 true로 설정되었을 때 N.alert의 overlay 요소가 context로 지정한 요소만큼 가려 줍니다.

When the modal option is set to true, the overlay element of N.alert covers as much as the element specified by context.

window 객체를 지정하면 화면 전체를 덮어 주고 jquery selector 나 jQuery object를 입력하면 지정한 요소만큼 만 가려줍니다.

If a window object is specified, the entire screen is covered, and if a jquery selector or jQuery object is entered, only the specified elements are covered.

입력 요소(select, input, textarea 등)를 지정하면 입력 요소 옆에 툴팁으로 메시지를 표시해 줍니다.

If an input element(select, input, textarea, etc.) is specified, a message is displayed as a tool tip next to the input element.
msg string|object undefined O

메시지 내용.

Message content.

메시지 문자열, jQuery object, HTML 문자열이나 HTML 요소를 지정할 수 있습니다.

You can specify message string, jQuery object, HTML string or HTML element.
vars array undefined X

메시지의 변수를 입력한 값으로 치환해 줍니다.

The variable of the message is replaced with the entered value.

메시지에 선언된 {index}와 같은 변수가 vars 옵션으로 설정 한 array의 index에 해당하는 값으로 치환됩니다.

Variables such as {index} declared in the message are replaced with the value corresponding to the index of the array set with the vars option.

N(window).alert({
    msg : "{0} {1}-JS.",
    vars : ["Hello", "Natural"]
}).show();

// Result message : "Hello Natural-JS"

If set to true, the HTML of the message is applied.

When set to true, the OK / Cancel button is displayed. When set to false, only the OK button is displayed.

The top(px) position of the message dialog.

The left(px) position of the message dialog.
width number|function 0 X

메시지 대화상자의 넓이.

The width of the message dialog.

  • number 타입으로 설정할 경우 입력한 수치(px)가 요소의 넓이로 설정됩니다.
  • If set to number type, the entered value(px) is set as the element width.

  • function 타입으로 설정할 경우 msgContext(modal 옵션이 true 일 때 화면에 덮는 요소), msgContents(메시지 콘텐츠 요소)가 인수로 전달되고 return 된 값으로 요소의 넓이가 설정됩니다.

    If set to function type, msgContext (the element that covers the screen when the modal option is true) and msgContents (the message content element) are passed as arguments, and the width of the element is set as the returned value.

    ...
// 대화상자의 넓이를 화면에 꽉 채우기// Fully fill the width of the dialog on the screen.
width : function(msgContext, msgContents) {
    return $(window).width();
},
...
height number|function 0 X

메시지 대화상자의 타이틀 영역을 제외한 콘텐츠의 높이.

The height of the content, excluding the title area of the message dialog.

  • number 타입으로 설정할 경우 입력한 수치(px)가 요소의 높이로 설정됩니다.
  • When set to number type, the entered value(px) is set as the element height.

  • function 타입으로 설정할 경우 msgContext(modal 옵션이 true 일 때 화면에 덮는 요소), msgContents(메시지 콘텐츠 요소)가 인수로 전달되고 return 된 값으로 요소의 높이가 설정됩니다.

    If set to function type, msgContext(the element that covers the screen when the modal option is true) and msgContents(the message content element) are passed as arguments, and the height of the element is set as the returned value.

    ...
// 대화상자의 높이를 화면에 꽉 채우기// Fully fill the height of the dialog on the screen.
height: function(msgContext, msgContents) {
    // 메시지 대화상자가 열릴 때 msgContents는 숨겨져 있기 때문에 show() 함수를 호출 한 다음 타이틀 영역의 높이를 가져와야 합니다.// You need to call the show () function and then get the height of the title area because the message Content is hidden when the message dialog opens.
    return $(window).height() - msgContents.show().find(".msg_title_box__").height();
},
...

closeMode string remove X

메시지 대화상자가 닫혔을 때 대화상자 요소를 감출 것 인지, 제거할 것 인지를 설정합니다.

Set whether to hide or remove the dialog element when the message dialog is closed.

  • hide : 메시지 대화상자 요소를 숨겨서 이전 상태를 유지합니다.
  • hide : Hides the message dialog element to maintain its previous state.
  • remove : 메시지 대화상자 요소를 제거하여 상태를 초기화합니다.
  • remove : Initializes the state by removing the message dialog element.
modal boolean true X

true로 설정하면 context로 지정한 요소를 가리는 overlay 요소를 생성하여 메시지 대화상자의 콘텐츠를 제외하고 모든 이벤트를 차단합니다.

If set to true, creates an overlay element that covers the element specified by context to block all events except the content of the message dialog.
