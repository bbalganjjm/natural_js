함수 Methods

Name Arguments Type Return Description
format N/A N/A array[object]

서식화를 실행합니다.

Executes the format.

  • 룰들을 객체 옵션으로 지정할 경우 : 서식화된 데이터 목록 객체를 반환합니다.

    If specifying rules as object options : Returns a formatted data list object.

  • data-validate 속성이 정의된 요소가 포함된 context 요소를 jQuery object로 지정할 경우 : 서식화된 데이터 값을 지정 한 요소에 보여 줍니다.

    If a context element containing an element with data-validate property defined is specified as a jQuery object : Shows the formatted data values in an element.
row number N/A

서식화할 데이터 목록의 행 index.

Row index of the list of data to be formatted.

row 인자가 설정되지 않은 경우 다음과 같이 작동합니다.

If the row argument is not set, it works as follows.

  • 룰들을 객체 옵션으로 지정할 경우 : 모든 행 데이터를 서식화합니다.

    If specifying rules as object options : Formats all row data.

  • data-validate 속성이 정의된 요소가 포함된 context 요소를 jQuery object로 지정할 경우 : 첫 번째 행 인덱스로 자동 지정됩니다.

    If a context element containing an element with data-validate property defined is specified as a jQuery object : It is automatically specified as the first row index.

룰들을 객체 옵션으로 지정했을 경우 다음과 같은 폼으로 데이터를 반환합니다.

If specifying rules as object options, it returns data in the following form :

[{
    "컬럼 프로퍼티 명" : "서식화된 데이터 값"
}]
[{
    "Column property name" : "Formatted data value"
}]
unformat N/A N/A array[object]

원래 데이터로 치환하거나 반환합니다.

Replaces or returns with original data.
row number N/A

데이터 목록에서 unformat 할 행의 index.

Index of the row to unformat in the data list.
key string N/A

행 데이터에서 unformat 할 컬럼 프로퍼티 명.

Name of the column property to be unformatted in the row data.
