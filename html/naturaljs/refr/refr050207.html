예제 Examples

1. Documents 인스턴스를 생성하고 페이지 추가하기.

1. Creating the Documents instance and add pages.

<section class="docsContainer"></section>

<script type="text/javascript">
       var docs = N(".docsContainer").docs();
       docs.add("ex-0001", "Example page", { url : "ex.html" });
</script>

2. Document에 파라미터 전달 하기

2. Passing parameters to the document

<section class="docsContainer"></section>

<script type="text/javascript">
       var docs = N(".docsContainer").docs();
       docs.request.attr("a", 1).request.attr("b", 2).add("ex", "예제", { url : "ex.html" });

       // ex.html의 Controller object의 init 함수의 두 번째 인수인 request 객체에서 request.attr("a");와 request.attr("b"); 구문을 사용해서 파라미터를 얻을 수 있습니다.
</script>
<section class="docsContainer"></section>

<script type="text/javascript">
       var docs = N(".docsContainer").docs();
       docs.request.attr("a", 1).request.attr("b", 2).add("ex", "예제", { url : "ex.html" });

       // You can get parameter with "request.attr('a');" and "request.attr('b');" syntax of request object, which is the second argument of init function of Controller object in ex.html.
</script>
