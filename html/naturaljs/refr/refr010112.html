N.json 객체의 함수 Function of N.json Object

N.json 은 json object 관련 유틸리티 함수들을 모아놓은 객체입니다.

N.gc is a collection object of utility functions related to json object.

"N.json.{함수명}(arg[0...N])"와 같이 사용할 수 있습니다.

You can use like "N.json.{function name}(arg[0...N])"

Name Arguments Type Return Description
mergeJsonArray N/A N/A array[json object]

json object가 담겨 있는 array 두 개를 병합합니다.

It merges two arrays containing json object.

arr1 인자를 기준으로 arr2 인자를 병합하고 중복된 요소를 제외합니다.

Merges the arr2 argument based on the arr1 argument and excludes duplicated elements.

세 번째 인자로 오브젝트의 프로퍼티명을 지정하면 해당 프로퍼티를 기준으로 중복된 요소를 제외합니다.

If the object property name is specified as the third argument, duplicate elements are excluded based on the property.

arr1 인자로 지정한 객체는 병합되더라도 메모리 참조는 변경되지 않습니다.

Memory references are not changed even if the object specified by the arr1 argument is merged.
arr1 array[json object] N/A

원본 array[json object]를 입력합니다.

Inputs the original array[json object].
arr2 array[json object] N/A

병합할 array[json object]를 입력합니다.

Inputs the array[json object] to be merged.
key string N/A

병합할 기준 프로퍼티명(필수 인자 아님)을 입력합니다.

Inputs the name of the base property to be merged(not a required argument).
format N/A N/A string

json object를 보기 좋게 formatting 해 줍니다.

Formats the json object beautifully.
obj array[json object]|json object N/A

json object 나 json object가 저장되어 있는 array를 입력합니다.

Inputs the json object or the array in which the json object is stored.
indent number N/A

들여 쓰기 될 공백 수를 입력합니다.

Inputs the number of spaces to be indented.
