예제 Examples

1. 메시지를 표시하기 위한 세 가지 방법

1. Three ways to display a message

<script type="text/javascript">
    // 옵션을 지정해서 메시지 표시
    N.notify(opts).add(msg, url);
    // 옵션 지정 없이 기본 옵션으로 메시지 표시
    N.notify.add(msg, url);
    // 메시지 대화 상자의 위치를 지정하고 옵션을 지정해서 메시지 표시
    N(position).notify(opts).add(msg, url);
</script>
<script type="text/javascript">
    // Displays the message by specifying options.
    N.notify(opts).add(msg, url);
    // Displays the message with default options without specifying options.
    N.notify.add(msg, url);
    // Specifies the position of the message dialog and displays the message by specifying options.
    N(position).notify(opts).add(msg, url);
</script>
