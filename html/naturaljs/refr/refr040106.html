예제 Examples

1. 지정한(.alertArea) 영역에 메시지 표시하기.

1. Displaying the message in the specified(.alertArea) area.

<div class="alertArea" style="height: 200px;border: 3px dotted var(--md-sys-color-outline-variant);"></div>

<script type="text/javascript">
       N(".alertArea").alert("Hello Natural-JS.").show();
</script>

2. 전체 화면(window)에 메시지 표시하기.

2. Displaying the message on full screen(window).

<script type="text/javascript">
       N(window).alert("Hello Natural-JS.").show();
</script>

3. 전체 화면(window)에 Confirm 메시지 대화 상자를 표시하고 드래그 앤 드롭 기능 활성화 하기

3. Displaying the Confirm message dialog on full screen and activate drag and drop function

<script type="text/javascript">
       N(window).alert({
           msg : "Hello Natural-JS.",
           confirm: true,
           draggable: true
       }).show();
</script>

4. 입력 요소 옆에 툴팁 메시지 표시하기.

4. Displaying the tooltip message next to input element.

<input class="alertInput" type="text" />

<script type="text/javascript">
       N(".alertInput").alert("Hello Natural-JS.").show();
</script>
