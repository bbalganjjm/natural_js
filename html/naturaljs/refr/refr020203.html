advisors 객체 advisors Object

natural.config.js 파일의 N.context.attr("architecture").cont.advisors 속성에 정의 합니다.

Define to N.context.attr("architecture").cont.advisors property of natural.config.js file

Name Type Default Required Description
pointcut object key string|object undefined O

advisor가 적용 될 pointcut을 정의 합니다.

Define pointcut to which advisor will be applied.

Controller object 의 함수명을 대상으로 지정 합니다.

Specifies the function name of the Controller object.

...
"pointcut" : {
    "type" : "regexp"
    "param" : "^init$"
}
...

위의 경우 pointcuts에서 regexp pointcut의 param 속성에 정의된 객체를 파라미터로 전달 합니다.

In the above case, the object defined in the param property of regexp pointcut is passed as a parameter in pointcuts.

...
"pointcut" : "^init$"
...

위와 같이 pointcut의 값이 객체가 아니고 regexp 문자열인 경우 내장 된 "regexp" pointcut을 기본값으로 사용 합니다.

As above, if the value of pointcut is not an object but a regexp string, the built-in "regexp" pointcut is used as the default.

pointcut param 속성 값 앞에 콜론(:) 으로 구분하여 view 요소를 식별 할 수 있는 jQuery selector 구문을 입력하면 선택 된 페이지만 적용 됩니다.

If you enter the jQuery selector syntax to identify the view element by separating it with a colon(:) before the pointcut param attribute value, only the selected page is applied.

...
"pointcut" : ".page01,#page02,.page03:^init$"
...
adviceType string undefined O

advisor 의 실행 시점을 설정 합니다.

Sets when to execute the advisor.

  • before : pointcut 으로 지정한 함수가 실행되기 전에 실행 됩니다.
  • before : Execute before executing the function specified in the pointcut.
  • after : pointcut 으로 지정한 함수가 실행 된 후에 실행 됩니다. pointcut 으로 지정한 함수의 반환값이 함께 전달 됩니다.
  • after : Execute after executing the function specified in the pointcut. The return value of the function specified by pointcut is also passed.
  • around : pointcut 으로 지정한 함수를 실행할 수 있는 joinPoint가 인자로 전달 됩니다.
  • around : Passed as an argument the joinPoint that can execute the function specified in the pointcut.
  • error : pointcut 으로 지정한 함수에서 예외가 발생 했을 때 실행 됩니다.
  • error : It is executed when an exception occurs in the function specified in the pointcut.
fn function undefined O

advisor 함수를 정의 합니다.

Define the advisor function.

adviceType 별 함수의 인자는 다음과 같습니다.

The arguments of functions by adviceType are as follows.


before

  • cont : Controller object
  • fnChain : 함수(명) 체인 문자열
  • fnChain : Function(name) Chain string
  • args : 원본 함수의 arguments
  • args : arguments of original function
...
"advisors" : [{
    "pointcut" : "^init$",
    "adviceType" : "before",
    "fn" : function(cont, fnChain, args) {
        // 페이지가 로딩 된 후 init 함수가 호출 되기 전에 실행 됩니다.
    }
}]
...
...
"advisors" : [{
    "pointcut" : "^init$",
    "adviceType" : "before",
    "fn" : function(cont, fnChain, args) {
        // After the page is loaded, it is executed before the init function is called.
    }
}]
...

after

  • cont : Controller object
  • fnChain : 함수(명) 체인 문자열
  • fnChain : Function(name) Chain string
  • args : 원본 함수의 arguments
  • args : arguments of original function
  • result : 원본 함수의 반환 값
  • result : Return value of original function
...
"advisors" : [{
    "pointcut" : "^init$",
    "adviceType" : "after",
    "fn" : function(cont, fnChain, args, result) {
        // 페이지가 로딩 된 후 init 함수가 호출 된 후에 실행 됩니다.
    }
}]
...
...
"advisors" : [{
    "pointcut" : "^init$",
    "adviceType" : "after",
    "fn" : function(cont, fnChain, args, result) {
        // After the page is loaded, it is executed after the init function is called.
    }
}]
...

around

joinPoint.proceed() 함수로 원본 함수를 실행 합니다.

Execute the original function with joinPoint.proceed () function.

  • cont : Controller object
  • fnChain : 함수(명) 체인 문자열
  • fnChain : Function(name) Chain string
  • args : 원본 함수의 arguments
  • args : arguments of original function
  • joinPoint : 원본 함수 실행 객체
  • joinPoint : Original function execution object
...
"advisors" : [{
    "pointcut" : "^init$",
    "adviceType" : "around",
    "fn" : function(cont, fnChain, args, joinPoint) {
        // 페이지가 로딩 된 후 init 함수가 호출 되기 전에 실행 됩니다.

        // before
        var result = joinPoint.proceed();
        // after
    }
}]
...
...
"advisors" : [{
    "pointcut" : "^init$",
    "adviceType" : "around",
    "fn" : function(cont, fnChain, args, joinPoint) {
        // After the page is loaded, it is executed before the init function is called.

        // before
        var result = joinPoint.proceed();
        // after
    }
}]
...

error

  • cont : Controller object
  • fnChain : 함수(명) 체인 문자열
  • fnChain : Function(name) Chain string
  • args : 원본 함수의 arguments
  • args : arguments of original function
  • e : errorThrown
...
"advisors" : [{
    "pointcut" : {
        "type" : "errorPointcut", // pointcuts 객체 설명의 errorPointcut 참고(무조건 허용)
        "param" : ""
    },
    "adviceType" : "error",
    "fn" : function(cont, fnChain, args, result, e) {
        // Controller object 에 정의한 함수에서 에러가 발생 할 때 마다 실행 됩니다.
    }
}]
...
...
"advisors" : [{
    "pointcut" : {
        "type" : "errorPointcut", // See errorPointcut in pointcuts object description(Unconditionally allowed)
        "param" : ""
    },
    "adviceType" : "error",
    "fn" : function(cont, fnChain, args, result, e) {
        // It is executed whenever an error occurs in the function defined in the Controller object.
    }
}]
...
