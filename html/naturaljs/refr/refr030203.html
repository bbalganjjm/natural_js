생성자 Constructor

Name Arguments Type Return Description
N.validator N/A N/A N.validator

N.validator 인스턴스를 생성합니다.

Create an N.validator instance.

var validator = N.validator(data, rules|context);
data array[object] N/A

유효성 검증할 데이터를 지정합니다.

Specifies the data to be validated.
rules|context object|jQuery object N/A

다음과 같이 두 가지 유형으로 유효성 검증 rule을 지정할 수 있습니다.

The validation rule can be specified in two types as follows.

  • 룰들을 객체 옵션으로 지정할 경우 :

    If specifying rules as object options : 

    // { "컬럼속성명" : [["룰명", arguments[0], arguments[1]... ] }
N.validator(data, {
    "numeric" : [["required"], ["integer+commas"]],
    "generic" : [["required"], ["korean"]],
    "limit" : [["required"], ["alphabet"]]
}.validate();
    // { "Column property name" : [["Rule name", arguments[0], arguments[1]... ] }
N.validator(data, {
    "numeric" : [["required"], ["integer+commas"]],
    "generic" : [["required"], ["korean"]],
    "limit" : [["required"], ["alphabet"]]
}.validate();

  • data-validate 속성이 정의된 요소가 포함된 context 요소를 jQuery object로 지정할 경우 :

    If a context element containing an element with data-validate property defined is specified as a jQuery object : 

    ...
<div class="validator-context">
    <!-- [ ["룰명", "룰arguments[0]", "룰 arguments[1]"], ... ] -->
    <input id="numeric" type="text" data-validate='[["required"], ["integer+commas"]]'/>
</div>
...

<script type="text/javascript">
N(".validator").cont({
    init : function(view, request) {
        N.validator(data, N(".validator-context", view)).validate();
    }
});
</script>
    ...
<div class="validator-context">
    <!-- [ ["Rule name", "arguments[0]", "arguments[1]"], ... ] -->
    <input id="numeric" type="text" data-validate='[["required"], ["integer+commas"]]'/>
</div>
...

<script type="text/javascript">
N(".validator").cont({
    init : function(view, request) {
        N.validator(data, N(".validator-context", view)).validate();
    }
});
</script>

    선언형으로 요효성 검증 룰 지정 시 동적으로 요효성 검증 룰을 변경하려면 다음과 같이 해당 입력 요소의 "validate" 데이터 속성 값에 요효성 검증 룰을 다시 지정하면 됩니다.

    If you want to change the validation rule dynamically when you specify the validation rule declaratively, you can specify the validation rule again in the "validate" data attribute value of the corresponding input element as follows.

    N("#numeric").data("validate", [["required"], ["integer"]])
N(obj).validator N/A N/A N.validator

N.validator의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

Create an object instance of N.validator with jQuery plugin method.

var validator = N(data).validator(rules|context);

객체 인스턴스를 생성하는 방식은 다르지만 "new N.validator()"로 생성한 인스턴스와 "N().validator"로 생성한 인스턴스는 동일합니다. N() 함수의 첫 번째 인자가 new N.validator 생성자의 첫 번째 인자로 설정됩니다.

The method of creating object instances is different, but the instance created with "new N.validator()" and the instance created with "N().validator" are the same. The first argument of the N () function is set as the first argument of the new N.validator constructor.
rules object|jQuery object N/A

N.validator 함수의 두 번째 인자(rules)와 같습니다.

Same as the second argument(rules) of the N.validator function.
