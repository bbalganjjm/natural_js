Name Type Default Required Description

Specifies the height of the toggle switch.

토글 스위치의 넓이를 지정 합니다.

Specifies the width of the toggle switch.

Specifies text when the toggle switch is activated.

토글 스위치가 비활성화됐을때의 텍스트를 지정합니다.

Specifies text when the toggle switch is inactivated.

토글 스위치의 텍스트 크기를 지정합니다.

소Specifies the text size for the toggle switch.

Switch 의 크기 관련 옵션은 특별히 지정하지 않는 한 `--njs-switch-height` 옵션 값을 기준으로 자동으로 계산됩니다.

Switch size-related options are automatically calculated based on the `--njs-switch-height` option value unless specifically specified.
