생성자 Constructor

Name Arguments Type Return Description
N.alert N/A N/A N.alert

N.alert 인스턴스를 생성합니다.

Create an N.alert instance.

var alert = N.alert(context, opts|msg);

var alert = N.alert(context, opts|msg, vars);
context window|jquery object|jquery selector string N/A

context 옵션을 설정 합니다. 기본 옵션의 context 옵션과 같습니다.

Set context options. Same as the context option of the default options.
opts|msg string|object N/A

string 타입의 인자를 입력하면 기본 옵션의 msg 옵션으로 설정 됩니다.

If you enter the argument of type string, it is set as the msg option of the default option.

msg 인자는 메시지 문자열, jquery object, HTML 문자열 이나 HTML 요소를 지정 할 수 있습니다.

The msg argument can specify a message string, jquery object, HTML string, or HTML element.

object 타입으로 인자를 입력 하면 컴포넌트의 기본 옵션 객체로 지정 됩니다.

If an argument is entered as the object type, it is specified as the components default option object.
vars array Alert

기본 옵션의 vars 옵션으로 설정 됩니다.

It is set as the vars option of the default option.
N(obj).alert N/A N/A N.alert

N.alert의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

Create an object instance of N.alert with jQuery plugin method.

var alert = N(context).alert(opts|msg);

var alert = N(context).alert(opts|msg, vars);

객체 인스턴스를 생성하는 방식은 다르지만 "new N.alert()" 로 생성한 인스턴스와 "N().alert" 로 생성한 인스턴스는 동일 합니다. N() 함수의 첫번째 인자가 new N.alert 생성자의 첫 번째 인자로 설정 됩니다.

The method of creating object instances is different, but the instance created with "new N.alert()" and the instance created with "N().alert" are the same. The first argument of the N () function is set as the first argument of the new N.alert constructor.
opts|msg string|object N/A

N.alert 함수의 두 번째 인자(opts|msg)와 같습니다.

Same as the second argument(opts|msg) of the N.alert function.
vars array N/A

N.alert 함수의 세번째 인자(vars)와 같습니다.

Same as the third argument(vars) of the N.alert function.
