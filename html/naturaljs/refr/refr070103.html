예제 Examples

1. 기본 토글 스위치 생성.

1. Create a basic toggle switch.

'input[type=checkbox]' 요소를 'label' 태그로 감싸고 class 속성값에 'switch__' 를 입력합니다. 그 다음 'input[type=checkbox]' 요소 다음에 'span' 태그를 추가하고 class 속성값에 'slider__' 를 입력 합니다.

Wrap the 'input[type=checkbox]' element with the 'label' tag and enter 'switch__' in the class attribute value. Then, add the 'span' tag after the 'input[type=checkbox]' element and enter 'slider__' in the class attribute value.

<label class="switch__">
    <input id="${id}" type="checkbox">
    <span class="slider__">
</label>

2. 스위치 옵션 지정.

2. Specify switch options.

'--njs-switch-width' 옵션을 '80px' 로 지정하여 가로로 넓은 토글 스위치를 만드는 예제 입니다.

This is an example of creating a horizontally wide toggle switch by setting the '--njs-switch-width' option to '80px'.

<label class="switch__" style="--njs-switch-width: 80px;">
    <input id="${id}" type="checkbox">
    <span class="slider__">
</label>

'--njs-switch-height' 옵션을 '70px' 로 지정하여 크기가 큰 토글 스위치를 만드고 안에 텍스트를 'Y/N' 으로 표시하는 예제 입니다.

This is an example of creating a large toggle switch by setting the '--njs-switch-height' option to '70px' and displaying the text inside as 'Y/N'.

<label class="switch__" style="--njs-switch-height: 40px; --njs-switch-on-content: 'Y'; --njs-switch-off-content: 'N';">
    <input id="${id}" type="checkbox">
    <span class="slider__">
</label>

Switch 의 크기 관련 옵션은 특별히 지정하지 않는 한 `--njs-switch-height` 옵션 값을 기준으로 자동으로 계산됩니다.

Switch size-related options are automatically calculated based on the `--njs-switch-height` option value unless specifically specified.
