팝업 컨텐츠 로딩이 완료 되면 팝업의 Controller object 에 caller 라는 속성이 생성됩니다. 이 caller 는 이 팝업을 생성한 부모페이지의 N.popup instance 입니다.
팝업에서 자체팝업을 닫을 때는 팝업의 Controller object 에서 "this.caller.close(onCloseData);" 명령을 실행 하면 됩니다. 이 명령을 실행하면 팝업이 닫히면서 onClose 이벤트 핸들러 함수의 첫번째 인자로 onCloseData 가 전달 됩니다.

When loading popup content is completed, a caller property is created in the controller object of the popup. This caller is the N.popup instance of the parent page that created this popup.
When closing a self-popup in a popup, in the controller object of the popup, "this.caller.close(onCloseData);" just run the command. When this command is executed, the popup is closed and onCloseData is passed as the first argument of the onClose event handler function.
