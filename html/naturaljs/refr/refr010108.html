N.date 객체의 함수 Function of N.date Object

N.date 은 날짜 제어 관련 유틸리티 함수들을 모아놓은 객체입니다.

N.gc is a collection object of utility functions related to date control.

"N.date.{함수명}(arg[0...N])"와 같이 사용할 수 있습니다.

You can use like "N.date.{function name}(arg[0...N])"

Name Arguments Type Return Description
diff N/A N/A string

입력 한 두 날짜 간 일수 차이를 반환합니다.

Returns the difference in days between two input dates.
refDateStr string N/A

기준 날짜를 8자리 날짜 문자열(YYYYmmdd)로 입력합니다.

Inputs the base date as an 8-digit date string (YYYYmmdd).
targetDateStr string N/A

계산할 날짜를 8자리 날짜 문자열(YYYYmmdd)로 입력합니다.

Inputs the date to be calculated as an 8-digit date string (YYYYmmdd).
strToDateStrArr N/A N/A array

입력된 날짜 문자열과 입력된 날짜 서식을 기준으로 array 타입의 날짜 객체[년(number|string), 월(number|string), 일(number|string)]를 생성합니다.

Creates an array type date object[year(number|string), month(number|string), day(number|string)] based on the input date string and input date format.
str string N/A

날짜 문자열을 입력합니다.

Inputs the date string.
format string N/A

입력한 날짜 문자열의 서식(format)을 입력(필수입력)합니다.

Inputs the format(required input) of the date string.

  • Y : 년
  • m : 월
  • d : 일
  • 예) "19991231" : "Ymd"
  • 예) "3112" : "dm"
  • 예) "12311999" : "mdY"
  • Y : year
  • m : month
  • d : day
  • Ex) "19991231" : "Ymd"
  • Ex) "3112" : "dm"
  • Ex) "12311999" : "mdY"
isString boolean N/A

true를 입력하면 결과 array 객체에 날짜를 number 타입이 아닌 string 타입으로 저장합니다.

If input is true, stores the date in the resulting array object as a string type, not as a number type.
strToDate N/A N/A string

입력된 날짜/시간 문자열을 date 객체로 변환하고 객체의 속성에 저장하여 반환합니다. 반환되는 객체의 속성은 다음과 같습니다.

Converts the input date/time string to a date object and stores it in the object's properties to return. The properties of the returned object are:

{
    obj : date object,
    format : Date format string
}
str string N/A

날짜 + 시간 문자열을 입력합니다.

Inputs the date + time string.

  • "19991231" : "1999-12-31 00:00:00"
  • "1999123103" : "1999-12-31 03:00:00"
  • "199912310348" : "1999-12-31 03:48:00"
  • "19991231034856" : "1999-12-31 03:48:56"
format string N/A

입력한 날짜 문자열의 서식(format)을 입력(필수입력)합니다.

Inputs the format(required input) of the date string.

  • Y : 년
  • m : 월
  • d : 일
  • H : 시
  • i : 분
  • s : 초
  • 예) "19991231" : "Ymd"
  • 예) "3112" : "dm"
  • 예) "12311999" : "mdY"
  • 예) "19991231120159" : "YmdHis"
  • Y : year
  • m : month
  • d : day
  • H : hour
  • i : minute
  • s : second
  • 예) "19991231" : "Ymd"
  • 예) "3112" : "dm"
  • 예) "12311999" : "mdY"
  • 예) "19991231120159" : "YmdHis"

format 인수를 입력하지 않으면 입력한 날짜 문자열의 길이에 따라 다음과 같이 자동으로 설정됩니다.

If the format argument is not input, it is automatically set as follows according to the length of the input date string.

  • 4자리 : "Y"
  • 6자리 : "Y-m"
  • 8자리 : "Y-m-d"
  • 10자리 : "Y-m-d H"
  • 12자리 : "Y-m-d H:i"
  • 14자리 : "Y-m-d H:i:s"
  • 4-digit : "Y"
  • 6-digit : "Y-m"
  • 8-digit : "Y-m-d"
  • 10-digit : "Y-m-d H"
  • 12-digit : "Y-m-d H:i"
  • 14-digit : "Y-m-d H:i:s"

대시(-), 콜론(:) 등의 날짜와 시간을 구분하는 문자는 Config(natural.config.js)의 N.context.attr("data").formatter.date 오브젝트의 함수들로 정의되어 있습니다. 날짜/시간 구분 문자는 해당 함수의 return 문자열로 변경할 수 있습니다.

Characters that separate date and time, such as dash(-) and colon(:), are defined as functions of N.context.attr("data").formatter.date object of Config(natural.config.js). The date/time separator can be changed to the return string of the corresponding function.
format N/A N/A array

입력된 날짜 + 시간 문자열을 입력된 날짜 서식을 기준으로 서식화(format)합니다.

Formats the input date + time string based on the input date format.
str string N/A

날짜 + 시간 문자열을 입력합니다.

Inputs the date + time string.
format string N/A

입력한 날짜 문자열의 서식(format)을 입력(필수입력)합니다.

Inputs the format(required input) of the date string.

  • Y : 년
  • m : 월
  • d : 일
  • H : 시
  • i : 분
  • s : 초
  • Y : year
  • m : month
  • d : day
  • H : hour
  • i : minute
  • s : second
N.date.format("19991231120159", "Y-m-d H:i:s") // "1999-12-31 12:01:59"
N.date.format("19991231120159", "Y/m/d H:i:s") // "1999/12/31 12:01:59"
N.date.format("19991231120159", "Y.m.d H:i") // "1999.12.31 12:01"
N.date.format("19991231120159", N.context.attr("data").formatter.date.YmdHis()) // "1999-12-31 12:01:59"

대시(-), 콜론(:) 등의 날짜와 시간을 구분하는 문자는 Config(natural.config.js)의 N.context.attr("data").formatter.date 오브젝트의 함수들로 정의되어 있습니다. 날짜/시간 구분 문자는 해당 함수의 return 문자열로 변경할 수 있습니다.

Characters that separate date and time, such as dash(-) and colon(:), are defined as functions of N.context.attr("data").formatter.date object of Config(natural.config.js). The date/time separator can be changed to the return string of the corresponding function.
dateToTs N/A N/A number

date 객체를 Timestamp로 변환합니다.

Convert a date object to a Timestamp.

dateObj 인수를 입력하지 않으면 현재 날짜와 시간으로 변환합니다.

If no dateObj argument is given, it is converted to the current date and time.
dateObj date object N/A

Timestamp로 변환할 date 객체를 입력합니다.

Inputs the date object to be converted to Timestamp.
tsToDate N/A N/A date object

Timestamp를 date 객체 변환합니다.

Convert a Timestamp to a date object.

tsNum 인수를 입력하지 않으면 현재 날짜와 시간으로 변환합니다.

If no tsNum argument is given, it is converted to the current date and time.
tsNum number N/A

date 객체로 변환할 Timestamp를 입력합니다.

Inputs the Timestamp to be converted to date object.
dateList N/A N/A array[array[date]]

연도 + 월별 날짜 객체 목록을 반환합니다.

Returns a list of date objects by year + month.
year number N/A

기준 연도를 입력합니다.

Inputs the base year.
month number N/A

기준 월을 입력합니다.

Inputs the base month.
