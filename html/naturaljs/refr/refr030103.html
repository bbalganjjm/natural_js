Name Arguments Type Return Description
N.formatter N/A N/A N.formatter

N.formatter 인스턴스를 생성합니다.

Create an N.formatter instance.

var formatter = N.formatter(data, rules|context);
data array[object] N/A

서식화할 데이터를 지정합니다.

Specifies the data to be formatted.
rules|context object|jQuery object N/A

다음과 같이 두 가지 유형으로 포맷 rule을 지정할 수 있습니다.

The format rule can be specified in two types as follows.

  • 룰들을 객체 옵션으로 지정할 경우 :

    If specifying rules as object options : 

    // { "컬럼속성명" : [["룰명", arguments[0], arguments[1]... ] }
N.formatter(data, {
    "numeric" : [["trimtoempty"], ["numeric", "#,###.##0000"]],
    "generic" : [["trimtoempty"], ["generic", "@@ABCD"]],
    "limit" : [["trimtoempty"], ["limit", "13", "..."]],
    "etc" : [["date", 12]]
}).format();
    // { "Column property name" : [["Rule name", arguments[0], arguments[1]... ] }
N.formatter(data, {
    "numeric" : [["trimtoempty"], ["numeric", "#,###.##0000"]],
    "generic" : [["trimtoempty"], ["generic", "@@ABCD"]],
    "limit" : [["trimtoempty"], ["limit", "13", "..."]],
    "etc" : [["date", 12]]
}.format();

  • data-validate 속성이 정의된 요소가 포함된 context 요소를 jQuery object로 지정할 경우 :

    If a context element containing an element with data-validate property defined is specified as a jQuery object : 

    ...
<div class="formatter-context">
    <!-- [ ["룰명", "arguments[0]", "arguments[1]"], ... ] -->
	<input id="limit" type="text" data-format='[["trimtoempty"], ["limit", "13", "..." ]]' />
</div>
...

<script type="text/javascript">
N(".formatter").cont({
    init : function(view, request) {
        N.formatter(data, N(".formatter-context", view)).format();
    }
});
</script>
    ...
<div class="formatter-context">
    <!-- [ ["Rule name", "arguments[0]", "arguments[1]"], ... ] -->
    <input id="limit" type="text" data-format='[["trimtoempty"], ["limit", "13", "..." ]]' />
</div>
...

<script type="text/javascript">
N(".formatter").cont({
    init : function(view, request) {
        N.formatter(data, N(".formatter-context", view)).format();
    }
});
</script>

선언형으로 포맷 룰 지정 시 동적으로 포맷 룰을 변경하려면 다음과 같이 해당 입력 요소의 "format" 데이터 속성 값에 포맷 룰을 다시 지정하면 됩니다.

If you want to change the format rule dynamically when you specify the format rule declaratively, you can specify the format rule again in the "format" data attribute value of the corresponding input element as follows.

N("#limit").data("format", [["trimtoempty"]])
N(obj).formatter N/A N/A N.formatter

N.formatter의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

Create an object instance of N.formatter with jQuery plugin method.

var formatter = N(data).formatter(rules|context);

객체 인스턴스를 생성하는 방식은 다르지만 "new N.formatter()"로 생성한 인스턴스와 "N().formatter"로 생성한 인스턴스는 동일합니다. N() 함수의 첫 번째 인자가 new N.formatter 생성자의 첫 번째 인자로 설정됩니다.

The method of creating object instances is different, but the instance created with "new N.formatter()" and the instance created with "N().formatter" are the same. The first argument of the N () function is set as the first argument of the new N.formatter constructor.
rules object|jQuery object N/A

N.formatter 함수의 두 번째 인자(rules)와 같습니다.

Same as the second argument(rules) of the N.formatter function.
