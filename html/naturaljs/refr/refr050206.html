유틸리티 Utility

Name Arguments Type Return Description
N.doc.createLoadIndicator N/A N/A N.docs

로딩바와 화면 클릭을 차단하는 요소를 원하는 지점에서 생성할 수 있습니다.

You can create elements that loading indicator and block the screen click at any point you want.

N.doc.createLoadIndicator 함수의 this를 call이나 apply 함수를 사용해서 N.docs 인스턴스로 지정해서 실행합니다.

Execute this by specifying "this" of N.doc.createLoadIndicator function as an N.docs instance using call or apply function.

var docsInst = N("#docsContainer").docs();
...
N.docs.createLoadIndicator.call(docsInst);

N.doc.createLoadIndicator와 N.doc.updateLoadIndicator, N.doc.removeLoadIndicator 함수를 override 하면 로딩바와 화면 클릭을 차단하는 요소를 다른 스타일로 표시할 수 있습니다.

By overriding the N.doc.createLoadIndicator, N.doc.updateLoadIndicator, and N.doc.removeLoadIndicator functions, elements that loading indicator and block the screen click can be displayed in different styles.
N.doc.updateLoadIndicator N/A N/A N.docs

로딩바의 진행률을 원하는 지점에서 업데이트할 수 있습니다.

You can update the progress of the loading indicator at any point you want.

N.doc.updateLoadIndicator 함수의 this를 call이나 apply 함수를 사용해서 N.docs 인스턴스로 지정해서 실행합니다.

Execute this by specifying "this" of N.doc.updateLoadIndicator function as an N.docs instance using call or apply function.

var docsInst = N("#docsContainer").docs();
...
N.docs.createLoadIndicator.call(docsInst);
...
...
N.docs.updateLoadIndicator.call(docsInst, 2, 2);
...
N.docs.updateLoadIndicator.call(docsInst, 1, 2);
...
N.docs.updateLoadIndicator.call(docsInst, 0, 2);
...

N.doc.createLoadIndicator와 N.doc.updateLoadIndicator, N.doc.removeLoadIndicator 함수를 override 하면 로딩바와 화면 클릭을 차단하는 요소를 다른 스타일로 표시할 수 있습니다.

By overriding the N.doc.createLoadIndicator, N.doc.updateLoadIndicator, and N.doc.removeLoadIndicator functions, elements that loading indicator and block the screen click can be displayed in different styles.

완료될 때까지 1 씩 감소합니다.
