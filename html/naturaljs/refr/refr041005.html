Name Arguments Type Return Description
data N/A N/A json object array|jQuery object[json object array]

컴포넌트에 바인딩되어 있는 가장 최근 상태의 데이터를 반환합니다.

Returns the data of the most recent state bound to the component.
selFlag boolean N/A

반환되는 데이터의 타입을 결정하는 구분 값입니다.

This is a division value that determines the type of data returned.

selFlag 인수 값에 따라 다음과 같은 데이터를 반환합니다.

The following data is returned according to the value of the selFlag argument.

  • undefined(selFlag 옵션이 설정되지 않은 경우) : json object array 타입의 데이터를 반환합니다.
  • undefined(When the selFlag option is not set) : Returns data of type json object array.

  • false : 컴포넌트에 바인딩되어 있는 jQuery object[json object array] 타입의 원래 유형의 데이터를 반환합니다.

    false : Returns the data of the original type of the jQuery object[json object array] type bound to the component.

    data 메서드로 가져온 데이터를 다른 데이터 관련 컴포넌트에 바인딩할 때는 반드시 "false"로 설정하여 원래 유형의 데이터를 바인딩해야 양방향 데이터 바인딩이 활성화됩니다.

    When binding data imported by the data method to other data-related components, you must set "false" to bind the original type of data to enable two way databinding.
context N/A N/A jQuery object

context 요소를 반환합니다.

Returns the context element.
selector string N/A

jQuery selector 구문을 입력하면 context 요소에서 지정한 요소를 찾아서 반환합니다.

If you input jQuery selector syntax, the element specified in the context element is found and returned.
bind N/A N/A N.pagination

context 옵션으로 지정한 요소에 데이터를 바인딩하여 Pagination을 생성합니다.

Pagination is created by binding data to the element specified by the context option.
data|totalCount array[object]|number N/A

바인딩할 data를 지정하거나 totalCount를 설정합니다.

Specifies the data to be bound or set totalCount.

인수 타입이 number 이면 totalCount로 설정되고 array 타입이면 data로 지정됩니다.

If the argument type is number, it is set to totalCount, and if it is an array type, it is specified as data.
totalCount number N/A

전체 행 수 값이 설정됩니다.

The total rows count value is set.
pageNo N/A N/A number|N.pagination

페이지 번호 값을 가져오거나 설정합니다.

Gets or sets the page number value.

인수가 없으면 pageNo 값을 가져오고 첫 번째 인수를 입력하면 입력된 값으로 pageNo가 설정됩니다.

If there is no argument, pageNo value is got, and if the first argument is entered, pageNo is set as the entered value.

N.pagination의 옵션 값만 변경되므로 이 함수를 실행한 후 bind 메서드를 실행해줘야 바뀐 값이 페이지네이션에 표시됩니다.

Only the option value of N.pagination changes, so after executing this function, the bind method must be executed to display the changed value in pagination.
pageNo number N/A

설정할 페이지 번호를 입력합니다.

Enters the page number to set.

currPageNavInfo N/A N/A N.pagination

페이징 정보 객체를 반환합니다.

Returns the paging information object.

currPageNavInfo 객체 정보

currPageNavInfo object information

  • pageNo : 현재 페이지 번호.
  • pageNo : Current page numbe.r
  • countPerPage : 페이지 당 행 수
  • countPerPage : Rows count per page.
  • countPerPageSet : 페이지 세트당 페이지 수.
  • countPerPageSet : Pages count per page set.
  • currSelPageSet : 현재 페이지 세트 번호.
  • currSelPageSet : Current page set number.
  • pageCount : 전체 페이지 수.
  • pageCount : Total pages count.
  • pageSetCount : 전체 페이지 세트 수.
  • pageSetCount : Total page sets count.
  • totalCount : 전체 행 수.
  • totalCount : Total rows count.
  • startPage : 현재 페이지 세트 중 첫 번째 페이지 번호.
  • startPage : Number of the first page in the current page set.
  • startRowIndex : 선택한 페이지의 첫 번째 행 인덱스.
  • startRowIndex : Index of the first row on the selected page.
  • startRowNum : 선택한 페이지의 첫 번째 행 번호
  • startRowNum : Number of the first row of the selected page.
  • endPage : 현재 페이지 세트 중 마지막 페이지 번호.
  • endPage : Number of the last page in the current page set.
  • endRowIndex : 선택한 페이지의 마지막 행 인덱스.
  • endRowIndex : Index of the last row on the selected page.
  • endRowNum : 선택한 페이지의 마지막 행 번호.
  • endRowNum : Number of the last row of the selected page.
