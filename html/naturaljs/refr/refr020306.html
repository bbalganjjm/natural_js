예제 Examples

1. 서버에서 데이터 조회

1. Query data from server

<script type="text/javascript">
    N.comm("data.json").submit(function(data) {
        N.log(data);
    });
</script>

2. 서버에 파라미터 전송 및 결과 데이터 조회

2. Send parameter to server and query result data

<script type="text/javascript">
    // N 함수의 인자가 파라미터로 전송 됩니다.
    N({ "param1": 1, "param2": "Mark" }).comm("data.json").submit(function(data) {
        N.log(data);
    });
</script>
<script type="text/javascript">
    // The arguments of the N function are sent as parameters.
    N({ "param1": 1, "param2": "Mark" }).comm("data.json").submit(function(data) {
        N.log(data);
    });
</script>

3. 지정한 요소에 HTML 페이지 삽입하기

3. Insert HTML page into specified element

<article class="view-0001"></article>

<script type="text/javascript">
    // ".view-0001" 요소에 "page.html" 페이지를 로드하고 "page.html" 페이지의 Controller object 의 init 메서드를 실행 시켜 줍니다.
    N(".view-0001").comm("page.html").submit(function() {
        // 페이지 삽입이 완료 된 후 실행 됩니다.
        // 필요 없으면 콜백 함수를 정의 하지 않아도 됩니다.
    });
</script>
<article class="view-0001"></article>

<script type="text/javascript">
    // Load "page.html" page in ".view-0001" element and execute init method of Controller object on "page.html" page.
    N(".view-0001").comm("page.html").submit(function() {
        // It executes after the page is inserted.
        // You don't need to define a callback function if you don't need it.
    });
</script>
