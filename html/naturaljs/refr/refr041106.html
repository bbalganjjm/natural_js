key, val, level, parent 옵션은 Config(natural.config.js)에 다음과 같이 전역 옵션으로 설정하면 N(data).tree("selector").bind();와 같이 간편하게 사용할 수 있습니다.

The key, val, level and parent options can be conveniently used as N(data).tree("selector").bind(); by setting them as global options in Config(natural.config.js) as follows:

N.context.attr("ui", {
    ...
    "tree": {
        "key": "key",
        "val": "val",
        "level": "level",
        "parent": "parent"
    }
    ...
});

다음 예제들은 Config(natural.config.js)에 다음과 같이 설정되어 있다는 가정하에 구동되는 예제입니다. 그렇지 않다면 컴포넌트 초기화 시 key, val, level, parent 옵션을 설정해야 합니다.

The following examples are based on the assumption that Config(natural.config.js) is configured as follows. Otherwise, you must set the key, val, level and parent options when initializing the component.

1. 기본 Tree 생성 하기.

1. Creating the default tree.

<div class="treeBlock"></div>

<script type="text/javascript">
    N(data).tree(".treeBlock").bind();
</script>

2. 체크박스 Tree 생성 하기.

2. Creating the checkbox tree.

<div class="treeBlock"></div>

<script type="text/javascript">
    N(data).tree({
        context: ".treeBlock",
        checkbox: true
    }).bind();
</script>
