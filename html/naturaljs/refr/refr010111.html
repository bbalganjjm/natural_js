N.array 객체의 함수 Function of N.array Object

N.array 은 배열 관련 유틸리티 함수들을 모아놓은 객체입니다.

N.gc is a collection object of utility functions related to array.

"N.array.{함수명}(arg[0...N])"와 같이 사용할 수 있습니다.

You can use like "N.array.{function name}(arg[0...N])"

Name Arguments Type Return Description
deduplicate N/A N/A array

array에서 중복된 요소를 제거합니다.

Removes duplicate elements from array.

array에 object가 담겨 있을 경우 두 번째 인수로 오브젝트의 프로퍼티명을 입력하면 입력한 프로퍼티를 기준으로 중복된 오브젝트를 제거합니다.

If an object is contained in the array, inputting the object's property name as the second argument removes the duplicated object based on the input property.
arr array N/A

중복을 제거할 array를 입력합니다.

Inputs the array to deduplicate.
key string N/A

array에 object가 저장되어 있을 경우 중복된 요소를 제거할 기준 프로퍼티명을 입력합니다.

If an object is stored in the array, inputs the base property name for remove the duplicate elements.
