jQuery 이벤트

jQuery Events

Datepicker의 context로 지정된 입력 요소에 다음과 같은 이벤트 핸들러를 바인딩할 수 있습니다.

You can bind the following event handler to the input element specified by the Datepicker 's context.

컴포넌트 초기화 시 같은 이름의 이벤트가 이미 바인딩되어 있다면 초기화 시 지정한 이벤트가 먼저 실행됩니다.

If an event of the same name is already bound at component initialization, the event specified at initialization is executed first.

Event Name Description
onSelect

날짜나 월(monthonly 옵션이 true로 설정 되었을 경우)이 선택 되었을 때 실행되는 이벤트 핸들러를 정의합니다.

Defines an event handler that is executed when a date or month(if the monthonly option is set to true) is selected.

$("selector").bind("onSelect", function(e, context, selDate, monthonly) {
    // e : 이벤트 객체
    // context : context 요소
    // selDate : 선택된 날짜의 Date 객체
    // monthonly : monthonly 옵션 값
});
$("context").bind("onSelect", function(e, context, selDate, monthonly) {
    // e : Event object
    // context : context element
    // selDate : The Date object of the selected date
    // monthonly : monthonly option value
});
onBeforeShow

데이트피커가 표시되기 전에 실행되는 이벤트 핸들러를 정의합니다.

Define an event handler that is executed before the date picker is displayed.

$("context").bind("onBeforeShow", function(e, context, contents) {
    // e : 이벤트 객체
    // context : context 요소
    // contents : Datepicker 판넬 요소
});
$("context").bind("onBeforeShow", function(e, context, contents) {
    // e : Event object
    // context : context element
    // contents : Datepicker panel element
});
onShow

데이트피커가 표시된 후에 실행되는 이벤트 핸들러를 정의합니다.

Define an event handler that is executed after the date picker is displayed.

$("context").bind("onShow", function(e, context, contents) {
    // e : 이벤트 객체
    // context : context 요소
    // contents : Datepicker 판넬 요소
});
$("context").bind("onShow", function(e, context, contents) {
    // e : Event object
    // context : context element
    // contents : Datepicker panel element
});
onBeforeHide

데이트피커가 닫히기 전에 실행되는 이벤트 핸들러를 정의합니다.

Defines an event handler that is executed before the datepicker closes.

$("context").bind("onBeforeHide", function(e, context, contents) {
    // e : 이벤트 객체
    // context : context 요소
    // contents : Datepicker 판넬 요소
});
$("context").bind("onBeforeHide", function(e, context, contents) {
    // e : Event object
    // context : context element
    // contents : Datepicker panel element
});
onHide

데이트피커가 닫힌 후에(닫히는 효과가 끝난 후) 실행되는 이벤트 핸들러를 정의합니다.

Define an event handler that is executed after the datepicker is closed(after the closing effect is over).

$("context").bind("onHide", function(e, context) {
    // e : 이벤트 객체
    // context : context 요소
});
$("context").bind("onHide", function(e, context) {
    // e : Event object
    // context : context element
});
onChangeYear

연도가 변경 되었을 때 실행되는 이벤트 핸들러를 정의합니다.

Defines an event handler that is executed when the year is changed.

$("context").bind("onHide", function(e, context, year, oe) {
    // e : 이벤트 객체
    // context : context 요소
    // year : 선택된 연도
    // oe : Original 이벤트 객체
});
$("context").bind("onHide", function(e, context, year, oe) {
    // e : Event object
    // context : context element
    // year : Selected year
    // oe : Original Event object
});
onChangeMonth

월이 변경 되었을 때 실행되는 이벤트 핸들러를 정의합니다.

Defines an event handler that is executed when the month is changed.

$("context").bind("onHide", function(e, month, year, oe) {
    // e : 이벤트 객체
    // context : context 요소
    // month : 선택된 월
    // year : 선택된 연도
    // oe : Original 이벤트 객체
});
$("context").bind("onHide", function(e, month, year, oe) {
    // e : Event object
    // context : context element
    // month : Selected month
    // year : Selected year
    // oe : Original Event object
});
