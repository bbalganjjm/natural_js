Documents.request

Document.request(N.docs.request)는 단독으로 사용될 수 없고 N.docs로 콘텐츠를 로딩할 마다 생성되는 객체입니다. request는 요청 정보를 갖고 있고 attr 메서드로 document 간 파라미터를 전달할 수 있습니다.

Document.request(N.docs.request) is an object that can't be used alone and is created every time content is loaded using N.docs. request has request information and can pass parameters between documents with attr method.

add 나 reload 함수를 호출하기 전에 다음과 같이 request.attr로 파라미터를 설정하면

If you set the parameter with request.attr as follows before calling add or reload function

var docs = N("container selector").docs();
docs.request.attr("pageParam", { "param" : 1 }).add("page id", "page name", { url : "url" });

docs.request.attr("pageParam", { "param" : 1 }).reload();

로드된 콘텐츠의 Controller object의 init 함수 request 인수로 파라미터를 얻을 수 있습니다.

You can get the parameter as the request argument of the init function of the Controller object on the loaded content.

N("page id").cont({
    init : function(view, request) {
        N.log(request.attr("pageParam")); // --> { a : 1 }
        // request 객체는 Controller object에서 this.request 속성으로도 참조할 수 있습니다.
    }
})

N("page id").cont({
    init : function(view, request) {
        N.log(request.attr("pageParam")); // --> { a : 1 }
        // The request object can also be referenced by the this.request property in the Controller object.
    }
})

Documents.request는 Communicator.request 모듈을 사용합니다. Documents.request에 대한 자세한 내용은 Communicator.request 메뉴를 참고 바랍니다.

Documents.request uses the Communicator.request module. For more information on Documents.request, please refer to the Communicator.request menu.
