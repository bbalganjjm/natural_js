생성자 Constructor

Name Arguments Type Return Description
N.cont N/A N/A Controller object

Controller object를 초기화 하고 Controller object를 반환합니다.

Initialize the Controller object and return the Controller object.

N.cont는 Controller object를 등록하고 관리 해 줍니다.

N.cont registers and manages the Controller object.

var controllerObject = N.cont(N(view), Controller object);
view jquery object N/A

view 영역을 jquery object 타입으로 지정 합니다.

Set the view area to jquery object type.
Controller object object N/A

Controller object를 정의 합니다.

Define the Controller object.

Controller object 에 init 함수를 생성 하면 view 로딩이 완료 되면 init 함수를 실행해 줍니다. init 함수의 인자로 view 요소와 Communicator.request 객체가 반환 됩니다.

If init function is created in the controller object, the init function is executed when the view loading is completed. The view element and Communicator.request object are returned as arguments to the init function.

N(".view").cont({
    init : function(view, request) {
    }
});

Controller object 에서 jQuery($|N) 로 요소를 선택할 때는 반드시 context 인자를 view 로 지정 해 줘야 합니다.

If selecting an element with jQuery($|N) in the Controller object, the context argument must be set to "view".

N(".view").cont({
    init : function(view, request) {
        N("selector", view);
    }
});
N().cont N/A N/A N/A

Controller object를 jQuery 플러그인 방식으로 초기화 합니다.

Initialize the Controller object with jQuery plugin method.

var controllerObject = N(view).cont(Controller object);

Controller object를 초기화 하는 방식은 다르지만 "new N.cont" 로 초기화 한 Controller object 와 "N().cont" 로 초기화 한 Controller object는 동일 합니다. N() 함수의 첫번째 인자가 new N.cont 생성자의 첫 번째 인자로 설정 됩니다.

The method of initializing Controller objects is different, but the Controller object initialized with "new N.cont" and the Controller object initialized with "N().cont" are the same. The first argument of the N () function is set as the first argument of the new N.cont constructor.
Controller object object N/A

N.cont 함수의 두 번째 인자(Controller object)와 같습니다.

Same as the second argument(Controller object) of the N.cont function.
