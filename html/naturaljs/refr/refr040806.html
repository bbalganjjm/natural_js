컴포넌트에 바인딩되어 있는 가장 최근 상태의 데이터를 반환합니다.

Returns the data of the most recent state bound to the component.
selFlag boolean|string N/A

반환되는 데이터의 타입을 결정하거나 데이터를 필터링 하기 위한 구분값입니다.

It is a division value to determine the type of data returned or to filter the data.

selFlag 인자 값에 따라 다음과 같은 데이터를 반환합니다.

The following data is returned according to the value of the selFlag argument.

  • undefined(selFlag 옵션이 설정되지 않은 경우) : array[json object] 타입의 데이터를 반환합니다.
  • undefined(When the selFlag option is not set) : Returns data of type array [json object].

  • false : 컴포넌트에 바인딩되어 있는 jQuery object[array[json object]] 타입의 원래 유형의 데이터를 반환합니다.

    false : Returns the data of the original type of the jQuery object[array[json object]] type bound to the component.

    data 메서드로 가져온 데이터를 다른 데이터 관련 컴포넌트에 바인딩할 때는 반드시 "false"로 설정하여 원래 유형의 데이터를 바인딩해야 양방향 데이터 바인딩이 활성화됩니다.

    When binding data imported by the data method to other data-related components, you must set "false" to bind the original type of data to enable two way databinding.

  • "modified" : 추가, 수정, 삭제된 데이터를 반환합니다(array[json object] 타입).
  • "modified" : Returns data that has been inserted, updated, or deleted.(array[json object] type).
  • "selected" : select 옵션이나 multiselect 옵션을 설정했을 때 선택된 데이터를 반환합니다(array[json object] 타입).
  • "selected" : Returns data selected when the select or multiselect option is set(array[json object] type).
  • "checked" : checkAll 옵션과 checkAllTarget 옵션이나 checkOnlyTarget 옵션을 설정했을 때 체크된 데이터를 반환합니다(array[json object] 타입).
  • "checked" : Checked data is returned when the checkAll option and checkAllTarget option or checkOnlyTarget option are set(array[json object] type).
  • "insert" : 추가된 데이터를 반환합니다(array[json object] 타입).
  • "insert" : Returns the inserted data(array[json object] type).
  • "update" : 수정된 데이터를 반환합니다(array[json object] 타입).
  • "update" : Returns the updated data(array[json object] type).
  • "delete" : 타입의 삭제된 데이터를 반환합니다(array[json object] 타입).
  • "delete" : Returns the deleted data(array[json object] type).

 string N/A

data 메서드의 두 번째 인자부터 n 번째 인자까지 데이터의 프로퍼티명을 인자로 지정하면 지정한 프로퍼티 값만 추출한 객체를 반환합니다.

If you specify the property name of the data from the second argument to the nth argument of the data method as an argument, the object that extracts only the specified property value is returned.

var listInst = N([]).list(".context")
    .bind([{ col01 : "", col02 : "", col03 : "", col04 : "", col05 : "", col06 : "" }]);
listInst.data("modified", "col01", "col02", "col03");
    // [{ col01 : "", col02 : "", col03 : "" }]

첫 번째 인자를 "modified", "selected", "checked", "insert", "update", "delete"로 설정했을 때만 작동됩니다.

It works only when the first argument is set to "modified", "selected", "checked", "insert", "update" or "delete".
context N/A N/A jQuery object

context 요소를 반환합니다.

Returns the context element.
selector string N/A

jQuery selector 구문을 입력하면 context 요소에서 지정한 요소를 찾아서 반환합니다.

If you input jQuery selector syntax, the element specified in the context element is found and returned.
contextBodyTemplate N/A N/A jQuery object

context 옵션으로 지정한 요소의 행요소(li)를 반환합니다.

Returns the row element(li) of the element specified by the context option.

반환된 요소를 수정 한 다음 bind 함수를 실행하면 수정된 요소로 행들이 생성됩니다.

If you modify the returned element and then execute the bind function, rows are created with the modified element.

contextBodyTemplate 함수를 사용하면 N.list가 초기화된 이후에도 N.list의 행 요소를 수정할 수 있습니다

The contextBodyTemplate function allows you to modify the N.list's row element even after N.list has been initialized.
selector string N/A

jQuery selector 구문을 입력하면 contextBodyTemplate 요소에서 지정한 요소를 찾아서 반환합니다.

If you input jQuery selector syntax, the element specified in the contextBodyTemplate element is found and returned.
bind N/A N/A N.list

context 옵션으로 지정한 요소 안에서 id 속성값을 갖고 있는 요소들에 데이터를 바인딩하고 data의 길이만큼 행 요소를 생성합니다.

Binds data to elements that have id attribute value in the element specified by context option and creates the row elements as long as the data length.
data array[json object] N/A

바인딩할 데이터를 지정합니다.

Specifies the data to be bound.
"append" string N/A

"append" 문자열을 인자로 설정하여 바인딩하면 이전에 바인딩된 데이터와 새로 바인딩 한 데이터를 병합하고 이전에 생성된 행 요소에 새로운 행 요소를 추가해 줍니다.

When binding by setting the "append" string as an argument, it merges the previously bound data with the newly bound data and appends the new row element to the previously created row element.

"append" 인자 값이 설정되면 scrollPaging.size 옵션 값 자동으로 0으로 지정되어 스크롤 페이징 기능이 비활성화됩니다.

When the "append" argument value is set, the scrollPaging.size option value is automatically set to 0 to disable the scroll paging feature.
add N/A N/A N.list

새로운 행 요소와 데이터를 추가합니다.

Adds new row elements and data.

행 데이터 생성 시 context 요소안에 있는 입력 요소들의 id 속성명과 값으로 데이터 객체를 생성합니다.

When creating row data, a data object is created with the id attribute name and value of the input elements in the context element.
data object|number N/A

data 객체를 지정하면 생성된 데이터와 지정 한 data가 병합되어 바인딩됩니다.

When a data object is specified, the added data and the specified data are merged and then bound.

data 인자가 number 타입이면 row(arguments[1]) 인자로 지정됩니다.

If the data argument is of type number, it is specified as a row (arguments [1]) argument.
row number N/A

새로운 데이터가 추가될 행 인덱스를 row 인자로 지정하면 지정한 행 인덱스 바로 앞에 행 데이터와 행 요소가 추가됩니다.

If the row index to which new data is to be added is specified as the row argument, the row data and row elements is added right in front of the specified row index.
remove N/A N/A N.list

행 요소와 데이터를 제거합니다.

Removes the row elements and data.
row number|array[number] N/A

제거할 행의 index를 지정합니다.

Specifies the index of the row to be removed.

행 index를 array에 담아 row 인자로 지정하면 한번에 여러 행을 제거합니다.

If you specify the row index in the array as a row argument, multiple rows are removed at once.
revert N/A N/A N.list

최초 바인딩된 초기 데이터나 add 시 생성된 초기 데이터로 되돌립니다.

Reverts to the initial bound data or initial data created when add.
row number|array[number] N/A

되돌릴 행 index를 지정합니다.

Specifies the row index to revert.

row 인자에 행 index를 지정하면 지정된 행 데이터만 되돌리고 row 인자를 입력하지 않으면 전체 행 데이터를 되돌립니다.

If a row index is specified for the row argument, only the specified row data is reverted. If no row argument is entered, the entire row data is reverted.

행 index를 array에 담아 row 인자로 지정하면 한번에 여러 행을 되돌립니다.

If you specify the row index in the array as a row argument, multiple rows are reverted at once.
validate N/A N/A N.list

추가 / 수정된 전체 데이터에 대한 유효성 검증 결과를 반환합니다. 유효성 검증 성공 시 true를 반환하고 실패 시 false가 반환 되면서 해당 입력 요소 옆에 툴팁으로 실패 메시지를 표시합니다.

Returns the result of validation on all added / modified data. If validation succeeds, it returns true, and on failure, false is returned, and a tooltip is displayed next to the input element.
row number N/A

row 인자에 행 index를 지정하면 지정된 행 데이터만 유효성 검증되고 row 인자를 입력하지 않으면 전체 행 데이터의 유효성이 검사됩니다.

If a row index is specified for the row argument, only the specified row data is validated. If no row argument is entered, the entire row data is validated.

컴포넌트에 바인딩되어 있는 데이터의 프로퍼티 값을 얻어오거나 설정합니다.

Gets or sets the property value of data bound to the component.

val 인자가 설정되면 지정된 값을 바인딩된 데이터에 세팅하거나 추가하고 val 인자가 설정되지 않으면 지정된 값을 얻어옵니다.

If the val argument is set, the specified value is set or added to the bound data. If the val argument is not set, the specified value is get.

If there is an element to which the data value is bound, null is converted to an empty string, number type is a numeric string, and boolean is converted to a "true" or "false" string and set. If there is no bound element, it is saved as the original type of the setting value.
select N/A N/A array

행을 선택하거나 선택된 행의 index를 반환합니다.

Selects the row or returns the index of the selected row.

첫 번째 인자(row)를 입력하지 않으면 선택된 행 들의 index 들을 반환하고 입력하면 지정된 행을 선택합니다.

If the first argument (row) is not entered, the indexes of the selected rows are returned, and if specified, the specified row is selected.

select 메서드를 사용하려면 select 나 multiselect 옵션이 true로 설정되어 있어야 합니다.

To use the select method, the select or multiselect option must be set to true.

If you specify the row index in the array as a row argument, multiple rows are selected at once.

입력하지 않거나 false를 입력하면 선택된 행들을 모두 선택해제한 후에 선택하고 true를 입력하면 기존 선택에 추가로 선택합니다.

Checks the elements specified by checkAllTarget and checkSingleTarget options or returns the index of the checked row.

첫 번째 인자(row)를 입력하지 않으면 체크된 행 들의 index 들을 반환하고 입력하면 지정된 행을 체크합니다.

입력하지 않거나 false를 입력하면 체크된 요소들을 모두 체크해제한 후에 체크하고 true를 입력하면 기존 체크에 추가로 체크합니다.

선택한 행 요소와 데이터를 이동합니다.

Moves the selected row element and data.

이동할 행의 index를 입력합니다.

Enters the index of the row to be moved.
toRow number N/A

이동할 위치의 행 index를 입력합니다.

Enter the row index of the position you want to move.

If the entered value is greater than the total length of the bound data, it moves to the next row of last row.
