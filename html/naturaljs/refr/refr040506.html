함수 Methods

Name Arguments Type Return Description
context N/A N/A jQuery object

context 요소를 반환합니다.

Returns the context element.
selector string N/A

jQuery selector 구문을 입력하면 context 요소에서 지정한 요소를 찾아서 반환합니다.

If you input jQuery selector syntax, the element specified in the context element is found and returned.
open N/A N/A N.tab|object

탭을 오픈 합니다.

Opens the tab.

인자들을 지정하지 않으면 다음과 같은 탭의 상태 정보가 object 타입으로 반환됩니다.

If no arguments are specified, the following tab status information is returned as the object type.

  • index : 활성화된 탭의 index
  • tab : 활성화된 탭 요소
  • content : 활성화된 탭 컨텐츠 요소
  • cont : 활성화된 탭 컨텐츠의 Controller object
  • index : Activated tab index
  • tab : Activated tab element
  • content : Activated tab content element
  • cont : Controller object of Activated tab content
index number N/A

index로 지정 한 탭과 탭 컨텐츠가 오픈 됩니다.

The tab and tab contents specified by index are opened.
onOpenData object N/A

페이지가 열릴 때마다 실행 되는 onOpen 이벤트 헨들러 함수의 첫 번째 인자로 전달할 데이터를 지정합니다.

Specifies the data to be passed as the first argument of the onOpen event handler function, which is executed whenever the page is opened.
enable N/A N/A N.tab

index로 지정한 탭을 활성화 합니다.

Enables the tab specified by index.
index number N/A

활성화할 탭 index를 입력합니다.

Inputs the tab index to be enabled.
disable N/A N/A N.tab

비활성화할 탭 index를 입력합니다.

Inputs the tab index to be disabled.
index number N/A

index로 지정 한 탭과 탭 컨텐츠가 오픈 됩니다.

The tab and tab contents specified by index are opened.
cont N/A N/A N.cont's object

탭 컨텐츠의 Controller object를 반환합니다.

Returns the Controller object of the tab content.

내부에서 생성된 탭 컨텐츠 이거나 preload 옵션이 false 이면 Controller object가 없는 상태이기 때문에 undefined가 반환됩니다.

If the tab content is created internally or the preload option is false, undefined is returned because there is no Controller object.
index number N/A

index 인자를 지정하지 않으면 활성화된 탭의 Controller object를 반환하고 index 인자를 지정하면 index로 지정한 탭 컨텐츠의 Controller object를 반환합니다.

If the index argument is not specified, the Controller object of the active tab is returned. If the index argument is specified, the Controller object of the tab content specified by index is returned.
