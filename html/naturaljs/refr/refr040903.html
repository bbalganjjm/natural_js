생성자 Constructor

Name Arguments Type Return Description
N.grid N/A N/A N.grid

N.grid 인스턴스를 생성합니다.

Create an N.grid instance.

var grid = N.grid(context, opts);
data array[json object] N/A

data 옵션을 설정 합니다. 기본 옵션의 data 옵션과 같습니다.

Set data options. Same as the data option of the default options.
opts object | jQuery Selector(string | jquery object) N/A

컴포넌트의 기본 옵션 객체를 지정 합니다.

Specifies the default option object for the component.

jQuery selector 문자열이나 jquery object를 입력하면 기본 옵션의 context 옵션으로 설정 됩니다.

Entering a jQuery selector string or jquery object sets it as the context option of default options.
N(obj).grid N/A N/A N.grid

N.grid의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

Create an object instance of N.grid with jQuery plugin method.

var grid = N(context).grid(opts);

객체 인스턴스를 생성하는 방식은 다르지만 new N.grid() 로 생성한 인스턴스와 N().grid 로 생성한 인스턴스는 동일 합니다. N() 함수의 첫번째 인자가 new N.grid 생성자의 첫 번째 인자로 설정 됩니다.

The method of creating object instances is different, but the instance created with newgrid and the instance created with grid are the same. The first argument of the N () function is set as the first argument of the new N.grid constructor.
opts object | jQuery Selector(string | jquery object) N/A

N.grid 함수의 두번째 인자(opts)와 같습니다.

Same as the second argument of the N.grid function.
